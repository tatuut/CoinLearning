## 🗣️ 対話ポイントの使い方

各Weekには **🗣️ 対話ポイント** があります。

### 例（Week 1の対話ポイント1）:

```markdown
## 🗣️ 対話ポイント1：銘柄を選ぼう！

**ミコ**: 「気になる銘柄を探してくれ」

👉 **あなたの番です**
```

ここで、Claude（私、ミコ）に話しかけてください:

```
例:
「ミコ、DOGEが気になる。一緒に見てくれる？」
```

すると、私が実データを取得して分析します。
