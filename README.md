# 🌿 草コイントレーダー

**100円→1000円への道**

草コイン（超低時価総額の仮想通貨）で少額から大きな利益を目指す、学習型トレーディングシステムです。

## ✨ 特徴

- 💰 **100円から始められる** - 少額でリスクを抑えながら学習
- 📚 **4週間の学習カリキュラム** - 基礎から実践まで段階的に学べる
- 🤖 **3つの取引戦略** - モメンタム、出来高急増、ブレイクアウト
- 📊 **詳細な分析機能** - 全ての取引を記録・分析して改善
- 🌍 **海外利用対応** - USDTやBTCに変換して世界中で使える

## 📁 プロジェクト構成

```

│

├── strategies/               # 取引戦略
│   ├── momentum.py          # モメンタム戦略
│   ├── volume_spike.py      # 出来高急増戦略
│   └── breakout.py          # ブレイクアウト戦略
│
├── analysis/                 # 分析ツール
│   └── performance.py       # パフォーマンス分析
│
└── curriculum/               # 学習カリキュラム
    ├── week1_basics.md      # Week 1: 基礎
    ├── week2_technicals.md  # Week 2: テクニカル分析
    ├── week3_strategy.md    # Week 3: 戦略構築
```

## 🚀 セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. データベースの初期化

```bash
python data/database.py
```

### 3. API設定（実際の取引をする場合）

```bash
# テンプレートをコピー
copy config\api_keys.json.template config\api_keys.json

# api_keys.json を編集してAPIキーを設定
```

**⚠️ 注意**: API KeyとSecretは絶対に公開しないこと！

## 💡 使用例

### 1. 市場スキャンで買いシグナルを探す

メインメニューから「1. 市場スキャン」を選択すると、3つの戦略で市場をスキャンして買いシグナルのあるコインを表示します。

### 2. 特定のコインを分析

気になるコイン（例: SHIBUSDT）を入力すると、全戦略で分析して総合判定を表示します。

### 3. 取引を記録

実際に取引したら「6. 取引を手動記録」で記録します。データベースに保存され、後で分析できます。

### 4. パフォーマンスを確認

「4. パフォーマンス分析」で、勝率、損益、100円→1000円への進捗などを確認できます。

## 🎯 100円→1000円への道

### 目標達成シミュレーション

**安定型（勝率60%）**:
- 平均利益 +20%
- 平均損失 -10%
- 約20-25回の取引で達成

**積極型（勝率40%）**:
- 平均利益 +50%
- 平均損失 -15%
- 約10-15回の取引で達成

**超慎重型（勝率80%）**:
- 平均利益 +5-10%
- 平均損失 -5%
- 約30-40回の取引で達成

## 🌍 海外での利用

### 仮想通貨を海外で使う方法

1. **草コイン → USDT に変換**
   - 利益が出たらUSDT（ステーブルコイン）に変換
   - 1 USDT = 1 USD で安定

2. **BTCに変換**
   - ビットコインは世界中で使える

3. **海外で現地通貨に換金**
   - Binance P2P
   - Crypto ATM
   - 現地の取引所

## ⚠️ リスクと注意事項

1. **投資は自己責任**: 必ず余剰資金で行う
2. **草コインは超ハイリスク**: 全額失う可能性もある
3. **感情的にならない**: ルールを守って冷静に
4. **少額から**: 100円など失っても困らない金額で
5. **学習が目的**: 最初は利益よりも経験を重視

## 📈 戦略詳細

### モメンタム戦略
- 上昇トレンドに乗る
- RSI、MACD、ROCを使用
- 中期的な値動きを狙う

### 出来高急増戦略
- 通常の3倍以上の出来高を検知
- 大きな値動きの前兆を捉える
- 短期取引向け

### ブレイクアウト戦略
- ボリンジャーバンドを使用
- レンジブレイクを狙う
- 大きな利益を狙う
