# 🔍 仮想通貨分析アシスタント

**「答えを出すツール」ではなく、「あなたが分析するための材料を一瞬で揃えるアシスタント」**

## 🎯 コンセプト

このツールは投資判断を自動化するものではありません。
**Claude Codeと一緒に分析するための材料を、ワンコマンドで取り寄せる**ためのツールです。

### できること

✅ 現在価格と24h統計を即座に取得
✅ 影響力の高いニュースを自動でスコアリング
✅ 価格とニュースの時系列を照らし合わせ
✅ スコアは「大雑把な概要把握」として提供
✅ 詳細分析は**実際のデータを見ながらClaude Codeと対話的に実施**

## 📦 使い方

### 基本コマンド

```bash
# BTCの全情報を取得
python crypto_analyst.py BTC

# ETHの全情報を取得
python crypto_analyst.py ETH

# SHIBの全情報を取得
python crypto_analyst.py SHIB
```

### 出力内容

1. **💰 現在の市場状況**
   - 現在価格
   - 24h変動率
   - 24h高値/安値
   - 出来高

2. **🎯 ニュース影響力スコア**
   - 最終スコア（大雑把な目安）
   - 関連性・重要性・影響力・時間減衰の内訳

3. **📰 最近のニュース（影響力順）**
   - タイトル、出典、公開日
   - 簡易スコアと本文プレビュー

4. **📉 30日間の価格動向**
   - 価格変動の概要
   - 最高値・最安値

### オプション機能

```bash
# 価格とニュースの時系列比較
python crypto_analyst.py BTC --timeline

# 特定ニュースの詳細表示（例: 1番目）
python crypto_analyst.py BTC --news 1
```

## 📊 データ構成

### スコアの見方

- **0.4以上**: 注目度が非常に高い
- **0.2～0.4**: 注目度が高い
- **0.1～0.2**: 中程度の注目
- **0.1未満**: 低注目

**重要**: スコアは「大雑把な目安」です。
実際の投資判断は、具体的なニュース内容と価格データを見て行ってください。

### ニュースの評価軸

1. **関連性** (Relevance): 銘柄との関連度
2. **重要性** (Importance): ニュース自体の重要度
3. **影響力** (Impact): 市場への影響力
4. **時間減衰** (Time Decay): 新しさによる重み付け

最終スコア = 関連性 × 重要性 × 影響力 × 時間減衰
