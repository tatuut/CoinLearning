# 📚 100円→1000円チャレンジ カリキュラム

実際に手を動かして100円→1000円を達成するための実践教材

---

## 🎯 このカリキュラムの特徴

### **対話型ハイブリッドアプローチ**

各Weekは3つのパートで構成：
1. **事前ストーリー**: 導入、セットアップ、技術解説
2. **🗣️ 対話ポイント**: あなたがClaude（ミコ）と一緒に分析
3. **Claude生成**: あなたのデータでカスタマイズされた解説

### **あなたが主人公**

- **登場人物**: あなた（tatut）とミコ（AIトレーダー）
- **実データで学習**: 架空のデータではなく、あなたの実際の取引データを使用
- **動的カリキュラム**: あなたの選択で内容が変わる

### **段階的学習**

```
Phase 1: 準備（Week 0）
   → 基礎知識を学ぶ

Phase 2: 実践開始（Week 1）
   → 100円で初めての購入

Phase 3: 深掘り（Week 2-4）
   → 実データで指標を理解

Phase 4: 高度な分析（Week 5-7）
   → 予測モデル、深層学習
```

## 🎓 各Weekの詳細

### Week 0: 基礎知識（買う前に学ぶ）

**内容**:
- ローソク足の4要素（始値・終値・高値・安値）
- RSI 70/30の意味
- MACDゴールデンクロス/デッドクロス
- ボリンジャーバンドの見方
- 損切り・分散投資の基本

**学べること**:
- 取引に必要な最低限の知識
- Week 1以降の前提知識

**前提**: なし

---

### Week 1: 初めての購入（100円チャレンジ）

**内容**:
- Coincheck登録（5,000円ボーナス獲得）
- MEXC API設定
- 🗣️ 対話: 気になる銘柄を選ぶ
- Claude生成: 実データでRSI/MACD分析
- 🗣️ 対話: 購入判断
- 購入実行
- 🗣️ 対話: 数日後の振り返り

**学べること**:
- Week 0の知識を実践で活用
- データに基づいた判断
- Claude（ミコ）との対話で理解を深める

**前提**: Week 0完了

---

### Week 2: RSI実践マスター

**内容**:
- 🗣️ 対話: Week 1の購入結果を報告
- Claude生成: 購入時と現在のRSI比較分析
- ワイルダーの発明ストーリー（1978年）
- RSIの計算方法とPython実装
- 🗣️ 対話: 次のアクション判断
- RSIダイバージェンス、期間調整
- 🗣️ 対話: 他の銘柄も分析

**学べること**:
- 自分のデータでRSIを深く理解
- なぜ70以上が「買われすぎ」なのか
- 成功/失敗の要因分析

**前提**: Week 1完了（購入済み）

---

### Week 3: MACD実践マスター

**内容**:
- 🗣️ 対話: 現在のポートフォリオ報告
- Claude生成: 全銘柄のMACD分析
- アペルの発明ストーリー（1970年代）
- MACDの計算方法とPython実装
- 🗣️ 対話: アクション判断（保持/損切り/新規購入）
- ヒストグラム変化、MACDダイバージェンス
- 🗣️ 対話: 新しい銘柄を探す

**学べること**:
- トレンドの方向と勢いの検出
- RSIとMACDの組み合わせ
- ゴールデンクロス/デッドクロスの実践

**前提**: Week 2完了

### Week 4: 統合テクニカル分析

**内容**:
- 🗣️ 対話: ポートフォリオの総合分析
- Claude生成: RSI+MACD+BBの統合スコア
- ボリンジャーの発明ストーリー（1980年代）
- 標準偏差と%B、バンド幅
- 統合スコアリングシステム
- 🗣️ 対話: 新しい銘柄を複数比較
- Claude生成: 統合スコアランキング
- 🗣️ 対話: 最終購入判断

**学べること**:

**学べること**:

- 総合的な判断ロジック
- 実践での使い方

**前提**: Week 5完了

## 📝 カリキュラム設計思想

### なぜ Week 0 から始まるのか？

```
旧カリキュラム:
Week 1: いきなり購入 → 失敗 → 後から学習

問題点:
- リスクが高い
- 失敗から学ぶのは非効率
- テクニカル指標を使わずに取引

新カリキュラム:
Week 0: 基礎知識習得 → Week 1: 理解して購入

メリット:
- リスク軽減
- 理由を持って買う
- Week 0の知識を実践で活用
```

### なぜ対話型なのか？

```
事前に書かれた詳細なストーリー:
- あなたの状況と合わない
- 架空のデータで学んでも身につかない

対話型ハイブリッド:
- あなたのデータで学ぶ
- Claude（ミコ）が個別化して解説
- 理解が深まる
```

### なぜ Week 2-4 で深掘りするのか？

```
Week 0: 基礎（RSI 70以上は買われすぎ）
Week 1: 実践（Week 0の知識で購入）
Week 2: 深掘り（なぜ70なのか？ダイバージェンスは？）

→ 段階的に理解が深まる
→ 実データで学ぶから忘れない
```
