# Week 2: テクニカル分析の実践 - データで勝つ技術

## このWeekの概要

Week 1では「材料を集めて理由を持って買う」ことを学びました。しかし、まだ「なんとなく」の部分が残っています。Week 2では、**数値で根拠を持つ**技術を学びます。RSI、MACD、ボリンジャーバンド、ボラティリティなどの技術指標を使い、「今が買い時か」「いつ売るべきか」を**データで判断**できるようになります。

ユウタは前週の成功に少し過信していますが、ミコが「運で勝っただけ」と指摘します。今週は、運ではなく**技術で勝つ**方法を体得します。

## 最終ゴール

このWeekが終わった時点でできること：

1. **`timeseries_storage.py`を使いこなす**
   - 任意の銘柄のRSI、MACD、ボリンジャーバンドを即座に確認できる
   - ボラティリティを見てリスクを判断できる

2. **技術指標を理解して使える**
   - RSI 30以下で「売られすぎ」、70以上で「買われすぎ」と判断
   - MACDのゴールデンクロス/デッドクロスを見つけられる
   - ボリンジャーバンドの±2σで買い/売りタイミングを判断

3. **データで振り返る習慣**
   - 取引後、必ず技術指標を確認
   - 「なぜ勝った/負けた」を数値で説明できる

4. **複数の指標を組み合わせる**
   - RSI + MACD + ボリンジャーバンドで総合判断
   - 1つの指標だけに頼らない

5. **リスクをコントロールする**
   - ボラティリティを見て取引額を調整
   - 高リスク銘柄では少額、低リスク銘柄では通常額

## 身につく知識・スキル

### 技術的知識
1. **RSI（相対力指数）**
   - 買われすぎ・売られすぎの判定
   - 0-100の範囲で表示
   - 30以下: 売られすぎ（買いチャンス）
   - 70以上: 買われすぎ（利確検討）

2. **MACD（移動平均収束拡散法）**
   - トレンドの転換点を見つける
   - ゴールデンクロス（上昇シグナル）
   - デッドクロス（下降シグナル）

3. **ボリンジャーバンド**
   - 価格の変動範囲を予測
   - ±2σの範囲に95%の確率で収まる
   - バンドの外 = 異常値（反発の可能性）

4. **ボラティリティ**
   - 価格変動の激しさ
   - 高ボラティリティ = ハイリスク・ハイリターン
   - 低ボラティリティ = ローリスク・ローリターン

### メンタル・マインドセット
1. **感覚より数値**
   - 「なんとなく」ではなく「RSIが32だから」

2. **複数の視点**
   - 1つの指標だけで判断しない

3. **継続的学習**
   - 毎回の取引で技術指標を確認
   - データで振り返り、次に活かす

### Part 2: 技術指標の説明（2,500文字目安）

#### 2.1 なぜ技術指標が必要か（400文字）

**ミコの説明**:
- 「材料（ニュース、価格、出来高）だけでは不十分」
- 「なぜなら、『今が買い時か』が分からないから」

**具体例**:
- 例1: ニュースが良くても、既に買われすぎている場合
  → RSIを見れば分かる
- 例2: 価格が上昇中でも、トレンド転換の兆候がある場合
  → MACDを見れば分かる
- 例3: 価格が異常に高い/低い場合
  → ボリンジャーバンドを見れば分かる

**技術指標の役割**:
- 「今が買い時/売り時かを数値で教えてくれる」
- 「感覚ではなく、客観的な判断ができる」

#### 2.2 RSI（相対力指数）の基礎（500文字）

**ミコの説明**:
- RSI = Relative Strength Index（相対力指数）
- 0-100の範囲で表示される
- **買われすぎ・売られすぎ**を判定する指標

**数値の意味**:
```
RSI 70以上: 買われすぎ（利確検討、新規買いは避ける）
RSI 30-70:  中立（通常の範囲）
RSI 30以下: 売られすぎ（買いチャンス）
```

**計算方法（簡易説明）**:
- 一定期間（通常14日）の価格上昇と下落を比較
- 上昇の勢いが強い → RSI高い
- 下落の勢いが強い → RSI低い

**実例で理解**:
```
BTC:
- 2025-10-14: RSI 65（買われ気味）
- 2025-10-17: RSI 28（売られすぎ）← 買いチャンス！
```

**注意点**:
- RSI単独では判断しない
- 他の指標と組み合わせる

#### 2.3 MACD（移動平均収束拡散法）の基礎（500文字）

**ミコの説明**:
- MACD = Moving Average Convergence Divergence
- **トレンドの転換点**を見つける指標
- 3つの要素: MACD線、シグナル線、ヒストグラム

**シグナルの種類**:
```
ゴールデンクロス（上昇シグナル）:
- MACD線がシグナル線を下から上に突き抜ける
- → 買いチャンス

デッドクロス（下降シグナル）:
- MACD線がシグナル線を上から下に突き抜ける
- → 売りシグナル
```

**見方**:
```
MACD > シグナル: 上昇トレンド
MACD < シグナル: 下降トレンド
ヒストグラム > 0: 勢いが強い
ヒストグラム < 0: 勢いが弱い
```

**実例**:
```
ETH:
- 10月15日: ゴールデンクロス発生 → 買い
- 10月17日: デッドクロス発生 → 売り
```

#### 2.4 ボリンジャーバンドの基礎（500文字）

**ミコの説明**:
- 価格の**変動範囲**を予測する指標
- 中央線（移動平均線）と上下のバンド
- バンドの幅 = ボラティリティ

**バンドの意味**:
```
±1σ（シグマ）: 約68%の確率で価格がこの範囲
±2σ: 約95%の確率で価格がこの範囲
±3σ: 約99.7%の確率で価格がこの範囲
```

**トレード戦略**:
```
価格が下限バンド（-2σ）に到達:
→ 売られすぎ、反発の可能性（買いチャンス）

価格が上限バンド（+2σ）に到達:
→ 買われすぎ、調整の可能性（利確検討）

バンド幅が狭くなる:
→ ボラティリティ低下、大きな動きの前兆
```

**注意点**:
- バンドの外に出ても、そのまま続くこともある
- 他の指標と併用

#### 2.5 ボラティリティの重要性（600文字）

**ミコの説明**:
- ボラティリティ = 価格変動の激しさ
- 高ボラティリティ = ハイリスク・ハイリターン
- 低ボラティリティ = ローリスク・ローリターン

**数値の目安**:
```
ボラティリティ < 10%:  低（安定）
ボラティリティ 10-20%: 中（通常）
ボラティリティ > 20%:  高（危険）
```

**リスク管理への応用**:
```
低ボラティリティ銘柄:
- 通常の取引額（100円）
- 利確: +15%, 損切り: -8%

中ボラティリティ銘柄:
- やや慎重に（100円）
- 利確: +20%, 損切り: -10%

高ボラティリティ銘柄:
- 少額（50円）または見送り
- 利確: +30%, 損切り: -15%
```

**ユウタの失敗を振り返る**:
```
PEPE（失敗した銘柄）:
- ボラティリティ: 25%（高い！）
- → 本来は少額または見送るべきだった
- ユウタはこれを見ずに100円投入
- 結果: -25%の損失
```

#### 3.3 出力の見方（500文字）

**ミコが解説**:

**1. 最新データセクション**:
- `close`: 終値（現在価格）
