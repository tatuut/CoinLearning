## 📚 学ぶこと

### 1. パフォーマンス分析

#### 重要な指標

**勝率**:
```
勝率 = (利益が出た取引 / 全取引) × 100%

例: 10回中6回利益 → 勝率60%
```

**平均利益率・損失率**:
```
平均利益 = 利益取引の平均%
平均損失 = 損失取引の平均%

理想: 平均利益 > 平均損失の2倍
```

**プロフィットファクター**:
```
PF = 総利益 / 総損失

1.5以上 = 良好
2.0以上 = 優秀
```

### 2. 自己分析

#### 振り返るべき質問

**利益が出た取引**:
- なぜ買ったのか？
- なぜそのタイミングだったのか？
- 指標は何を示していたか？
- 再現可能か？

**損失が出た取引**:
- なぜ負けたのか？
- ルールを守ったか？
- 感情的になっていないか？
- 次はどう改善する？

### 3. 戦略の改善

#### PDCAサイクル

```
Plan（計画）
  ↓
Do（実行）
  ↓
Check（評価）
  ↓
Act（改善）
  ↑________↓
```

**実践例**:
```
Plan: RSI戦略で5回取引
Do: 実際に取引を実行
Check: 勝率3/5、平均利益+15%、平均損失-8%
Act: RSI 30→25に調整、損切り-10%→-7%に変更
```

### 5. よくある失敗と対策

#### 失敗1: リベンジトレード
```
❌ 損失後に取り戻そうと大きく賭ける
✅ ルール通りに小額で次の取引
```

#### 失敗2: 利確できない欲
```
❌ 「もっと上がる」と思って持ち続ける
✅ 利確ルールを守る
```

#### 失敗3: 損切りできない恐怖
```
❌ 「戻るかも」と思って持ち続ける
✅ 損切りラインで即売却
```

#### 失敗4: FOMO買い
```
❌ 既に急騰したコインを高値で買う
✅ 次のチャンスを待つ
```
