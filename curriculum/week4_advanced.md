## 🌍 Lesson: 海外で使う方法

### ミコの次の教え

**ミコ**: 「で、ユウタ。この1027円、どうする？」

**ミコ**: 「最初に言ってたよな。『海外で使いたい』って」

**ユウタ**: 「そうだ！」

**ミコ**: 「じゃあ、**USDT（ステーブルコイン）**に変えよう」

#### USDTとは

```
USDT = Tether
1 USDT = 1 USD（ドル）

【メリット】
✅ 価格が安定（ドルと連動）
✅ 世界中で使える
✅ 換金が簡単
```

**ミコ**: 「草コインのまま持ってると、価格が変動する。でもUSDTなら安定してる」

**ユウタ**: 「なるほど！変換する！」

---

### 海外での使い方

**ミコ**: 「海外でUSDTを使う方法は3つ」

```
【方法1: Binance P2P】
USDTを現地の人に売って現地通貨をもらう
→ 手数料ほぼゼロ

【方法2: Crypto ATM】
USDTを現地通貨に換金できるATM
→ 手数料 5-10%

【方法3: 現地取引所】
USDTを現地取引所で売る
→ 手数料 1-3%
```

**ユウタ**: 「マジで世界中で使えるんだ...」

**ミコ**: 「そう。だから仮想通貨は強い」
