## 📚 学ぶこと

### 1. 出来高急増を狙う戦略

**出来高 = 取引されている量**

```
出来高が急増 = 大きな値動きの前兆
```

**Volume Spike戦略**:
```
1. 通常の10倍以上の出来高を検知
2. 即座に購入（早い者勝ち）
3. +20-30%で利確 or -10%で損切り
```

**リスク**: 急騰後に急落することも多い

### 2. ニュース連動取引

**草コインはニュースで激変する**

**例**:
- 取引所上場発表 → 🚀 急騰
- 有名人のツイート → 🚀 急騰
- プロジェクト進捗発表 → 📈 上昇
- ハッキング/詐欺ニュース → 📉 暴落

**戦略**:
```
1. Twitter/Reddit/Telegramを監視
2. ポジティブニュースで即購入
3. 短期で利確（数時間〜1日）
```

### 3. モメンタム戦略

**勢いに乗る戦略**

```
価格が上昇中 → さらに上がる可能性
（順張り）
```

**条件**:
- 24時間で+50%以上上昇
- 出来高も増加中
- RSI 60-80（まだ買われすぎじゃない）

### 4. リスク管理の鉄則

#### ルール1: 損切りラインを必ず設定
```
購入価格の -10% で自動売却
→ 損失を限定する
```

#### ルール2: 利確ラインも設定
```
+30%で半分売却
+50%で全部売却
→ 利益を確定する
```

#### ルール3: ポートフォリオ分散
```
1つのコインに全額投資しない
3-5種類に分散
```

#### ルール4: FOMO（取り残される恐怖）を避ける
```
「もう遅い」と思ったら買わない
次のチャンスを待つ
```

```

```

**メリット**:
- 為替手数料なし
- 銀行不要
- 世界中どこでも使える
