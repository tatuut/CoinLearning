### 📝 Week 1.5以降（動的に追加）

Week 1.5以降の教材は、**実践で必要になったものから追加**されます。

#### 追加のトリガー

```
実践中に遭遇
    ↓
「この状況、今の知識じゃ分からない！」
    ↓
新しい指標・概念を学ぶ
    ↓
新しいWeekまたはコラムを追加
```

#### 候補となる指標（MEXC表示可能）

- **トレンド系**: MA, EMA, SAR, ICHIMOKU（一目均衡表）
- **オシレーター系**: KDJ, WR, PSY, CCI, DMI
- **ボリューム系**: VOL, OBV, VR
- **その他**: BIAS, BRAR, CR, DMA, TRIX, EMV, ROC, MTM

## 🎯 教材の使い方

### 1. 初めての方

```
1. Week 0 を読む（基礎知識）
    ↓
2. Week 1 を読む（初めての購入）
    ↓
3. 実際に100円で取引してみる
    ↓
4. CONCEPT_TREE.mdで習得状況をチェック
```

### 2. 復習したい方

```
1. CONCEPT_TREE.mdで未習得項目を確認
    ↓
2. 該当するWeekの教材を復習
    ↓
3. 理解できたらチェック ✅
```

### 3. 新しい指標を学びたい方

```
1. 実践で「この指標が必要だ」と気づく
    ↓
2. CONCEPT_TREE.mdの候補リストを確認
    ↓
3. 実践で試してみる
    ↓
4. 必要なら新しいWeekを作成（または依頼）
```
