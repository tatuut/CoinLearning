**サヤ** - 完全初心者（22歳、大学生）
投資経験ゼロ。わからないことはすぐ質問する。納得するまで聞く性格。

**tatut**（あなた） - これから始める人

## 📚 Part 1: ローソク足の読み方

### Scene 1: チャートって何？

**サヤ**: 「ミコ、チャートってグラフでしょ？なんであんな棒みたいなのがいっぱいあるの？」

**ミコ**: 「あれは『ローソク足』って言うんだ。1本で価格の動きがわかる」

**サヤ**: 「1本で？どういうこと？」

**ミコ**: 「例えば、今日のBTCの値動きを見てみよう」

```
今日のBTC（1日足）:
朝9時（始値）: 100,000ドル
昼12時: 105,000ドルまで上昇（高値）
午後3時: 98,000ドルまで下落（安値）
夜9時（終値）: 102,000ドル
```

**ミコ**: 「この4つの値段を、1本のローソク足で表現するんだ」

**サヤ**: 「4つも？どうやって？」

**ミコ**: 「こう描く」

```
    │  ← 上ヒゲ（高値 105,000まで）
  ┌─┐
  │ │ ← 実体（始値 100,000 → 終値 102,000）
  └─┘
    │  ← 下ヒゲ（安値 98,000まで）
```

**サヤ**: 「えっと...箱の下が始値で、上が終値？」

**ミコ**: 「正解！そして、ヒゲが『どこまで動いたか』を示してる」

**サヤ**: 「上ヒゲが長いってことは...高値までいったけど、戻されたってこと？」

**ミコ**: 「完璧だ。売り圧力が強かったってことだな」

---

### Scene 2: 陽線と陰線

**サヤ**: 「でもさ、下がった日はどう描くの？」

**ミコ**: 「いい質問だ。終値が始値より安い日は『陰線』になる」

**サヤ**: 「さっきのは『陽線』？」

**ミコ**: 「そう。終値 > 始値なら陽線。色は緑とか白」

```
陽線（上がった日）:
終値 ┌─┐
    │ │  上がった！
始値 └─┘

陰線（下がった日）:
始値 ┌─┐
    │ │  下がった...
終値 └─┘
```

**サヤ**: 「じゃあ、陰線ばっかりなら下降トレンド？」

**ミコ**: 「その通り。陽線ばっかりなら上昇トレンド」

**サヤ**: 「簡単じゃん！」

**ミコ**: 「...と思うだろ？実はヒゲも重要なんだ」

**サヤ**: 「ヒゲ？さっきの？」

---

### Scene 3: ヒゲの意味

**ミコ**: 「例えば、こんなローソク足があったら？」

```
   │││  ← すごく長い上ヒゲ
  ┌┐
  │││
  └┘
```

**サヤ**: 「うわ、上ヒゲ長っ！」

**ミコ**: 「これ、何が起きたと思う？」

**サヤ**: 「えーっと...一度すごく上がったけど、めっちゃ下げられた？」

**ミコ**: 「正解。買いが入って上がったけど、売りが殺到して押し戻された」

**サヤ**: 「ってことは...売りの勢いが強い？」

**ミコ**: 「そういうこと。これは『売りシグナル』かもしれない」

**サヤ**: 「じゃあ、下ヒゲが長いのは？」

**ミコ**: 「逆だ。一度下がったけど、買い戻された。『買いシグナル』の可能性」

**サヤ**: 「なるほど...ヒゲは『戦いの痕跡』みたいなものか」

**ミコ**: 「いい表現だな。買いと売りの戦いが見えるんだ」

---

### Scene 4: 練習問題

**ミコ**: 「じゃあ、これはどう？」

```
   │
 ┌─┐
 │█│  ← 長い陽線、短い上ヒゲ
 └─┘
   │
```

**サヤ**: 「えーっと...陽線だから上がった。上ヒゲ短いから...高値圏で終わってる？」

**ミコ**: 「完璧。これは？」

**サヤ**: 「買いの勢いが強い！上昇トレンド継続！」

**ミコ**: 「正解だ。じゃあこれは？」

```
   │
 ┌─┐
 │█│  ← 長い陰線、短い下ヒゲ
 └─┘
   │
```

**サヤ**: 「陰線で...下ヒゲ短い...売りの勢いが強い！下降トレンド！」

**ミコ**: 「素晴らしい。もうローソク足は読めるな」

**サヤ**: 「でもさ、ローソク足だけじゃ不十分でしょ？他に何見るの？」

**ミコ**: 「次は『RSI』だ」

---

## 📊 Part 2: RSI（相対力指数）

### Scene 5: RSIって何？

**サヤ**: 「RSI...アールエスアイ？」

**ミコ**: 「Relative Strength Index。『買われすぎ・売られすぎ』を0-100の数字で示す」

**サヤ**: 「0-100？テストの点数みたい」

**ミコ**: 「そう考えていい。70点以上は『買われすぎ』、30点以下は『売られすぎ』」

**サヤ**: 「待って。買われすぎって...買われすぎたら何がダメなの？」

**ミコ**: 「いい質問だ。買われすぎってことは...？」

**サヤ**: 「みんな買ってる...もう買う人がいない...？」

**ミコ**: 「そう！買う人がいなくなったら、価格は...？」

**サヤ**: 「下がる！」

**ミコ**: 「正解。だから『RSI 70以上 = 調整（下落）の可能性』」

---

### Scene 6: でも、なんで70なの？

**サヤ**: 「ちょっと待って。なんで70なの？60じゃダメ？80じゃダメ？」

**ミコ**: 「おお、いい疑問だ」

**ミコ**: 「RSIを発明したワイルダーって人が、過去のデータを分析して『70を超えたら調整が入ることが多い』って発見したんだ」

**サヤ**: 「過去のデータ？」

**ミコ**: 「そう。何千回も取引を調べて、統計的に『70』が良いって結論を出した」

**サヤ**: 「じゃあ絶対じゃないんだ」

**ミコ**: 「そう。あくまで『確率が高い』ってだけ。だから他の指標と組み合わせる」

**サヤ**: 「なるほど...」

---

### Scene 7: 実例で理解

**ミコ**: 「じゃあ、実際の例を見よう」

```
BTCのRSI:
今日: RSI 75
```

**サヤ**: 「75！70超えてる！買われすぎ！」

**ミコ**: 「今、BTCを買う？」

**サヤ**: 「えーっと...買われすぎだから、そろそろ下がる？だから買わない？」

**ミコ**: 「正解。調整を待つ。RSIが60以下になってから検討」

**サヤ**: 「じゃあ、RSI 25だったら？」

**ミコ**: 「どう思う？」

**サヤ**: 「25...30以下だから売られすぎ！買いのチャンス！」

**ミコ**: 「そういうこと。ただし...」

**サヤ**: 「ただし？」

**ミコ**: 「さらに下がる可能性もある。だから他の指標も見る」

**サヤ**: 「他の指標...MACDとか？」

**ミコ**: 「そうだ。次はMACDを学ぼう」

---

## 📈 Part 3: MACD（移動平均収束拡散）

### Scene 8: MACDって何の役に立つの？

**サヤ**: 「MACD...マックディー？」

**ミコ**: 「マックディーって読む。『トレンドの方向と強さ』を教えてくれる」

**サヤ**: 「RSIは『買われすぎ・売られすぎ』で、MACDは『トレンド』？」

**ミコ**: 「正解。役割が違うんだ」

**サヤ**: 「じゃあ、RSIだけじゃダメなの？」

**ミコ**: 「例えば、RSI 65だったら？」

**サヤ**: 「65...中立？買ってもいい？」

**ミコ**: 「でも、トレンドが下降中だったら？」

**サヤ**: 「あ...下降トレンドなら、買ったら下がる...」

**ミコ**: 「そう。だからMACDで『今、上昇中か下降中か』を確認する」

---

### Scene 9: ゴールデンクロスとデッドクロス

**ミコ**: 「MACDには2本の線がある。MACD線とSignal線」

**サヤ**: 「2本？」

**ミコ**: 「MACD線がSignal線を下から上に抜けたら...」

**サヤ**: 「上に抜けた？買いシグナル？」

**ミコ**: 「正解。これを『ゴールデンクロス』って呼ぶ」

**サヤ**: 「じゃあ、上から下に抜けたら？」

**ミコ**: 「どう思う？」

**サヤ**: 「売りシグナル...『デッドクロス』？」

**ミコ**: 「完璧だ」

**サヤ**: 「待って、なんで『クロス』なの？」

**ミコ**: 「2本の線が交差（クロス）するからだ」

```
ゴールデンクロス:
     MACD線
    ╱
   ╱  ← 下から上に抜ける！
  ╱
━━━━━━ Signal線

デッドクロス:
━━━━━━ Signal線
  ╲
   ╲  ← 上から下に抜ける
    ╲
     MACD線
```

**サヤ**: 「なるほど...視覚的にわかりやすい」

---

### Scene 10: RSI + MACDで判断

**ミコ**: 「じゃあ、組み合わせてみよう」

```
銘柄A:
RSI: 65（中立）
MACD: ゴールデンクロス発生
```

**サヤ**: 「えーっと...RSIは中立だから買われすぎじゃない。MACDはゴールデンクロスで上昇トレンド開始...買い？」

**ミコ**: 「正解！2つの指標が『買い』を示してる」

**サヤ**: 「じゃあこれは？」

```
銘柄B:
RSI: 75（買われすぎ）
MACD: ゴールデンクロス発生
```

**サヤ**: 「うーん...RSIは買われすぎだけど、MACDは上昇トレンド...どっち？」

**ミコ**: 「矛盾してるよな。こういう時は...？」

**サヤ**: 「様子見？」

**ミコ**: 「そう。矛盾シグナルは危険。RSIが60以下に下がるのを待つ」

**サヤ**: 「なるほど...両方が一致してる時だけ動く」

**ミコ**: 「賢いな」

---

## 📉 Part 4: ボリンジャーバンド

### Scene 11: ボリンジャーバンドって何？

**サヤ**: 「次は...ボリンジャーバンド？」

**ミコ**: 「価格の『正常範囲』を示すバンドだ」

**サヤ**: 「正常範囲？」

**ミコ**: 「例えば、BTCが普段100,000ドル前後で動いてるとする」

**サヤ**: 「うん」

**ミコ**: 「ある日、いきなり150,000ドルになったら？」

**サヤ**: 「異常！買われすぎ！」

**ミコ**: 「その『正常か異常か』を自動で計算してくれるのがボリンジャーバンド」

**サヤ**: 「便利じゃん！」

---

### Scene 12: バンドの見方

**ミコ**: 「バンドは3本の線がある」

```
━━━━━━━━  ← 上限バンド
～～～～～～  ← 中心線
━━━━━━━━  ← 下限バンド
```

**サヤ**: 「真ん中が普通で、上限超えたら買われすぎ？」

**ミコ**: 「正解。下限下回ったら？」

**サヤ**: 「売られすぎ！買いのチャンス！」

**ミコ**: 「そういうこと」

**サヤ**: 「じゃあ、RSIと同じじゃん」

**ミコ**: 「似てるけど違う。ボリンジャーバンドは『ボラティリティ』も教えてくれる」

**サヤ**: 「ボラティリティ？」

**ミコ**: 「価格の変動幅だ」

---

### Scene 13: バンドの広がり

**ミコ**: 「バンドが広がってる時は...」

```
━━━━━━━━━━━  ← 広い
    ╱╲
   ╱  ╲      ← 激しく動いてる
  ╱    ╲
━━━━━━━━━━━
```

**サヤ**: 「価格が激しく動いてる？」

**ミコ**: 「そう。チャンスでもあり、リスクでもある」

**サヤ**: 「じゃあ、バンドが狭い時は？」

```
━━━━━━━━━━━  ← 狭い
   ─────     ← 静か
━━━━━━━━━━━
```

**ミコ**: 「価格が静か。でも...」

**サヤ**: 「でも？」

**ミコ**: 「静かな後は、大きく動くことが多い」

**サヤ**: 「嵐の前の静けさ！」

**ミコ**: 「まさに。これを『スクイーズ』って呼ぶ」

---

### Scene 14: 3つの指標を組み合わせる

**サヤ**: 「ねえ、結局どうやって使うの？全部？」

**ミコ**: 「そう。3つ全部見る」

```
例:
RSI: 28（売られすぎ）
MACD: ゴールデンクロス（上昇開始）
BB: 下限付近（売られすぎ）

→ 3つとも『買い』 → 強い買いシグナル
```

**サヤ**: 「3つが一致したら信頼度高い！」

**ミコ**: 「正解。逆に、矛盾してたら...？」

**サヤ**: 「様子見！」

## 🛡️ Part 5: リスク管理

### Scene 15: 損切りって何？

**サヤ**: 「ねえ、もし買った後に下がったら？」

**ミコ**: 「いい質問だ。『損切り』するんだ」

**サヤ**: 「損切り？損を切る？」

**ミコ**: 「そう。損失を限定するために、早めに売る」

**サヤ**: 「でも、上がるまで待てばいいじゃん」

**ミコ**: 「待ってたら...？」

**サヤ**: 「上がる...かも？」

**ミコ**: 「もっと下がるかもしれない」

**サヤ**: 「あ...」

---

### Scene 16: 損切りラインの決め方

**ミコ**: 「購入価格の-8〜-10%で損切りする」

**サヤ**: 「なんで8%？」

**ミコ**: 「例えば、1000円で買ったとする」

**サヤ**: 「うん」

**ミコ**: 「-10%は900円。-20%は800円。-50%は500円」

**サヤ**: 「...500円はキツイ」

**ミコ**: 「そう。早めに切れば、損失は小さい。次のチャンスに備えられる」

**サヤ**: 「なるほど...でも、どこで決めるの？」

**ミコ**: 「購入**前**に決める」

**サヤ**: 「購入前？」

**ミコ**: 「買った後に決めると、感情で判断しちゃう」

**サヤ**: 「『もう少し待てば...』って？」

**ミコ**: 「そう。だから先にルールを決める」

---

### Scene 17: 分散投資

**サヤ**: 「あと、全額1つの銘柄に突っ込むのはダメなんでしょ？」

**ミコ**: 「よく知ってるな」

**サヤ**: 「だって、それが失敗したら全部なくなるじゃん」

**ミコ**: 「正解。だから分散する」

```
総資金 10,000円:
✅ 良い例:
5銘柄 × 2,000円ずつ
→ 1銘柄失敗しても -2,000円

❌ 悪い例:
1銘柄 × 10,000円
→ 失敗したら -10,000円
```

**サヤ**: 「わかりやすい！」
