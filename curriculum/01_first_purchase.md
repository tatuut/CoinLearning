### Mission 2: 入金（XLM使用）

#### XLM（Stellar）を使う理由

XRPは最低20 XRP（約6,300円）必要なので、500円では送金できません。
代わりに**XLM（Stellar）**を使います。

**XLMのメリット**:
- 最低送金額: 0.01 XLM（約0.42円）
- 送金手数料: 0.01 XLM（約0.42円）
- 500円で購入可能: 約11.9 XLM

#### 入金手順

1. Coincheckで500円分のXLM（Stellar）を購入
2. MEXCにXLMを送金
3. MEXCでXLMをUSDTに交換

**詳細な手順は既存ドキュメント参照**:
- [MEXC API設定ガイド](../docs/MEXC_API_SETUP.md)

**ミコ**: 「プロトレーダーはみんな使ってる。一緒に設定しよう」

#### APIセットアップ手順

**詳細は別ドキュメントを参照**:
👉 **[MEXC API設定ガイド](../docs/MEXC_API_SETUP.md)**

**概要**:
1. MEXCにログイン → API管理ページへ
2. API Keyを作成
   - API名: `grass-coin-trader`
   - 取引権限: ✅ 有効
   - 出金権限: ❌ **絶対に無効**（セキュリティ）
   - IPアドレス制限: ✅ 有効（自宅IPのみ）
3. API KeyとSecretをコピー
4. 環境変数に保存

**Windowsの場合**:
```powershell
# PowerShellで実行
[System.Environment]::SetEnvironmentVariable('MEXC_API_KEY', 'あなたのAPIキー', 'User')
[System.Environment]::SetEnvironmentVariable('MEXC_API_SECRET', 'あなたのAPIシークレット', 'User')
```

**確認**:
```powershell
echo $env:MEXC_API_KEY
```

**ミコ**: 「API Secretは絶対に誰にも見せるな。漏れたら資産が盗まれる」

**あなた**: 「...わかった」
