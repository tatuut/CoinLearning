# 🌳 概念ツリー - 習得チェックリスト

**最終更新**: 2025-11-03

このファイルは、トレーディングに必要な**すべての概念をTree構造**で管理します。

---

## 📋 使い方

### ✅ チェックの意味

- `[x]` - 習得済み（理論を理解し、実践で使える）
- `[ ]` - 未習得（まだ学んでいない）
- `[~]` - 学習中（理論は分かったが実践経験不足）

### 🎯 学習のルール

1. **順番は自由** - 必要になったものから学ぶ
2. **動的に追加** - 新しい概念が出てきたら追加する
3. **実践で確認** - 理論だけでなく実践で使えるようになったらチェック


## 🎯 次に学ぶべき概念（推奨）

### フェーズ1（現在）の課題

実践の中で以下の課題に遭遇したら、対応する概念を学びましょう：

| 課題 | 学ぶべき概念 |
|-----|------------|
| 「トレンドの方向が分からない」 | MA（移動平均線） |
| 「いつ買えばいいか分からない」 | エントリータイミング |
| 「出来高が重要らしいけど...」 | VOL, OBV |
| 「指値注文って何？」 | 注文方法（指値） |
| 「過去データで検証したい」 | バックテスト |

### フェーズ2-3で必要になる概念

**1000円達成後**に以下を学びます：

| 課題 | 学ぶべき概念 |
|-----|------------|
| 「このプロジェクトは本物か？」 | ファンダメンタルズ分析（プロジェクト・チーム） |
| 「トークノミクスが分からない」 | トークノミクス（供給量、配分、インフレ率） |
| 「界隈の盛り上がりを知りたい」 | センチメント分析（SNS、コミュニティ） |
| 「インフルエンサーの信頼性は？」 | インフルエンサー分析 |
| 「バイラルする可能性は？」 | バイラル性分析 |

**重要**: フェーズ2-3の概念は、フェーズ1（テクニカル基礎）を習得してから学びましょう
