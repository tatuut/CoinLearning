## 🕯️ Part 1: ローソク足の読み方

### ローソク足とは？

**ローソク足（キャンドルスティック）** は、一定期間の価格の動きを1本の「ローソク」で表現したものです。

### ローソク足の構成要素

```
    │  ← 上ヒゲ（高値まで）
  ┌─┐
  │ │ ← 実体（始値〜終値）
  └─┘
    │  ← 下ヒゲ（安値まで）
```

| 要素 | 意味 |
|------|------|
| **始値（Open）** | 期間開始時の価格 |
| **終値（Close）** | 期間終了時の価格 |
| **高値（High）** | 期間中の最高価格 |
| **安値（Low）** | 期間中の最安価格 |

### 陽線と陰線

#### 陽線（緑/白）- 上昇
```
終値 ┌─┐
    │ │  上がった！
始値 └─┘
```
- 終値 > 始値
- 買いの勢いが強い

#### 陰線（赤/黒）- 下落
```
始値 ┌─┐
    │ │  下がった...
終値 └─┘
```
- 終値 < 始値
- 売りの勢いが強い

### ヒゲの意味

**上ヒゲが長い**:
- 一度は高値をつけたが、押し戻された
- 売り圧力が強い

**下ヒゲが長い**:
- 一度は安値をつけたが、買い戻された
- 買い圧力が強い

### 実践: ローソク足を読む

#### 例1: 強気のサイン
```
   │
 ┌─┐
 │█│  ← 長い陽線 + 短い上ヒゲ
 └─┘
   │
```
**解釈**: 買いが強く、上昇トレンド

#### 例2: 弱気のサイン
```
   │
 ┌─┐
 │█│  ← 長い陰線 + 短い下ヒゲ
 └─┘
   │
```
**解釈**: 売りが強く、下降トレンド

#### 例3: 迷い
```
   │
 ┌┐
 │││  ← 短い実体 + 長い上下ヒゲ
 └┘
   │
```
**解釈**: 方向感なし、様子見

## 📊 Part 2: RSI（相対力指数）

### RSIとは？

**RSI（Relative Strength Index）** = 「買われすぎ」「売られすぎ」を0-100の数値で示す指標

### RSIの見方

```
100 ━━━━━━━━━━━━━━━━━
 70 ━━━━━━━━━━━━━━━━━  ← 買われすぎゾーン

 50 ━━━━━━━━━━━━━━━━━  ← 中立

 30 ━━━━━━━━━━━━━━━━━  ← 売られすぎゾーン
  0 ━━━━━━━━━━━━━━━━━
```

| RSI値 | 意味 | アクション |
|-------|------|-----------|
| **70以上** | 買われすぎ | 売却検討（調整の可能性） |
| **30-70** | 中立 | 様子見 |
| **30以下** | 売られすぎ | 購入検討（反発の可能性） |

### 実例で理解

#### ケース1: RSI 75（買われすぎ）
```
あなた: 「今、BTCのRSIが75です。買うべき？」
答え: 「待った方が良い。調整が入る可能性が高い」
```

**理由**:
- RSI 70超え = 短期的に買われすぎ
- 価格が下がる可能性が高い
- 待ってRSIが60以下になってから検討

#### ケース2: RSI 25（売られすぎ）
```
あなた: 「ETHのRSIが25に下がりました」
答え: 「買いのチャンス！反発する可能性あり」
```

**理由**:
- RSI 30以下 = 売られすぎ
- 価格が反発する可能性が高い
- ただし、さらに下がるリスクもある

### RSIの落とし穴

⚠️ **注意**: RSIだけで判断しない！

- 強いトレンド中はRSI 70以上が続くことも
- 他の指標（MACD、ボリンジャーバンド）と併用

## 📈 Part 3: MACD（移動平均収束拡散）

### MACDとは？

**MACD** = トレンドの方向と強さを示す指標

### MACDの構成

```
    MACD線（青）
    ↓
━━━━━━━━━━━━━━
    ↑
    Signal線（オレンジ）

━━━━━━━━━━━━━━  ← ゼロライン
```

| 要素 | 意味 |
|------|------|
| **MACD線** | 短期と長期の移動平均の差 |
| **Signal線** | MACDの移動平均 |
| **Histogram** | MACD - Signal の差 |

### MACDの見方

#### ゴールデンクロス（買いシグナル）
```
     MACD
    ╱
   ╱  ← 上抜け！買い！
  ╱
━━━━━━ Signal
```
- MACD線がSignal線を**下から上**に抜ける
- 上昇トレンドの始まり

#### デッドクロス（売りシグナル）
```
━━━━━━ Signal
  ╲
   ╲  ← 下抜け！売り！
    ╲
     MACD
```
- MACD線がSignal線を**上から下**に抜ける
- 下降トレンドの始まり

### Histogramの見方

```
Histogram拡大
 ║║║║║  ← トレンド強化
  ║║║
   ║
━━━━━━━━━
```
- Histogram拡大 = トレンド強化
- Histogram縮小 = トレンド弱化

### 実例で理解

#### ケース1: ゴールデンクロス発生
```
昨日: MACD < Signal（下にいた）
今日: MACD > Signal（上抜けた！）
→ 買いシグナル！
```

#### ケース2: デッドクロス発生
```
昨日: MACD > Signal（上にいた）
今日: MACD < Signal（下抜けた...）
→ 売りシグナル！
```

## 📉 Part 4: ボリンジャーバンド

### ボリンジャーバンドとは？

**Bollinger Bands** = 価格の変動範囲を示すバンド

### バンドの構成

```
━━━━━━━━  ← 上限バンド（+2σ）

～～～～～～  ← 中心線（移動平均）

━━━━━━━━  ← 下限バンド（-2σ）
```

### ボリンジャーバンドの見方

| 位置 | 意味 | アクション |
|------|------|-----------|
| **上限に接触** | 買われすぎ | 売却検討 |
| **中心付近** | 通常の範囲 | 様子見 |
| **下限に接触** | 売られすぎ | 購入検討 |

### バンドの広がり

#### バンド拡大 = ボラティリティ高
```
━━━━━━━━━━━  ← 広がっている
    ╱╲
   ╱  ╲      ← 価格変動大
  ╱    ╲
━━━━━━━━━━━
```
- 価格が大きく動いている
- チャンス？リスク？

#### バンド収縮 = ボラティリティ低
```
━━━━━━━━━━━  ← 狭くなっている
   ─────     ← 価格変動小
━━━━━━━━━━━
```
- 価格が横ばい
- **大きな動きの前兆**かも

### 実例で理解

#### ケース1: 下限タッチ
```
現在価格: 下限バンドに接触
→ 「売られすぎ。反発の可能性」
```

#### ケース2: バンド収縮後の拡大
```
昨日: バンド幅狭い（静か）
今日: バンド拡大開始！
→ 「大きな動きが始まる予兆」
```

## 🛡️ Part 5: リスク管理

### 草コイン投資の鉄則

#### 1. 余剰資金のみ投資

```
給料 100,000円
  ├─ 生活費: 70,000円
  ├─ 貯金: 20,000円
  └─ 投資: 10,000円 ← これだけ！
```

**ルール**: 無くなっても生活に影響しない金額のみ

#### 2. 損切りラインの設定

```
購入価格: 100円
損切りライン: 90円（-10%）
↓
90円になったら自動売却
```

**理由**: 損失を限定する

#### 3. 分散投資

```
❌ 悪い例:
全額10,000円 → 1銘柄（SHIB）のみ

✅ 良い例:
10,000円 → 5銘柄に分散
  ├─ SHIB: 2,000円
  ├─ DOGE: 2,000円
  ├─ PEPE: 2,000円
  ├─ FLOKI: 2,000円
  └─ BONK: 2,000円
```

**理由**: 1銘柄が下がっても他でカバー

#### 4. ポジションサイジング

**初心者ルール**:
```
総資金の10-20%まで
例: 総資金10,000円なら
1銘柄は1,000-2,000円まで
```

### 感情のコントロール

#### やってはいけないこと

❌ **FOMO（取り残される恐怖）**
```
みんな: 「SHIB爆上げ中！」
あなた: 「乗り遅れる！今すぐ買わなきゃ！」
→ 高値掴み...
```

❌ **ナンピン（下がったら買い増し）**
```
100円で購入
↓
80円に下落
「安くなった！倍買おう！」
↓
60円にさらに下落...
```

✅ **やるべきこと**

```
1. 計画を立てる
2. ルールを守る
3. 冷静に分析
4. 感情で動かない
```
