**目標**: 理論武装してから投資を始める

**所要時間**: 3.5時間

**前提**: なし（完全初心者向け）

---

**ユウタ（23歳）** - あなたと一緒に草コインに挑戦する主人公
- 職業: 大学4年生（就活中）+ コンビニバイト
- 経済状況:
  - バイト代: 月8万円
  - 家賃: 5万円
  - 食費: 2万円
  - 光熱費: 1万円
  - 残り: 雀の涙（貯金ゼロ）
- 動機: 「海外旅行とか行きたいのに、金がないんだよな...」
- 性格: 好奇心旺盛だが焦りやすい。学ぶことには前向き

**ミコ** - AIトレーダー（Claude Code）
- 正体: 仮想通貨取引を学習したAI
- 過去: かつて感情に流されて資金を溶かした経験あり
- 今: データと理論で冷静に判断できるようになった
- 口調: 簡潔で的確。厳しいが優しい
- 特技: データ分析、市場調査、パターン認識

    ┃   終値（終了時の価格）

**ユウタ**: 「えーと...高値、安値、始値、終値の4つ？」

**ミコ**: 「完璧だ」

**ユウタ**: 「でも、これだけで何が分かるの？」

```

    ┃ ← 高値: $1.05
 ┏━┓

```

**ワイルダー**: 「価格そのものじゃダメだ...」

**ワイルダー**: 「必要なのは...**相対的な強さ**だ」

---

**【数式の誕生】**

*ペンを走らせる。*

```
【ワイルダーの思考プロセス】

Step 1: 上昇と下落を分離
- 上昇幅 = 今日の終値 - 昨日の終値（正の時のみ）
- 下落幅 = 昨日の終値 - 今日の終値（負の時のみ）

Step 2: 平均を取る
- 平均上昇幅 = 過去14日間の上昇幅の平均
- 平均下落幅 = 過去14日間の下落幅の平均

Step 3: 比率を計算
- RS (Relative Strength) = 平均上昇幅 / 平均下落幅

問題: RSは0～∞の範囲。見づらい...

Step 4: 0-100に正規化
- RSI = 100 - (100 / (1 + RS))

→ これだ！
```

## Part 4: MACD - トレンドの方向を見極める（45分）

### Scene 6: ジェラルド・アペルの挑戦

**ユウタ**: 「RSIで買われすぎが分かった。次は？」

**ミコ**: 「トレンドを見る。上昇トレンドか、下降トレンドか」

**ユウタ**: 「それって重要なの？」

**ミコ**: 「当たり前だ。下降トレンドで買ったら負ける」

**ミコ**: 「そのトレンドを見る技術を発明した男がいる。ジェラルド・アペル、1970年代のことだ」

---

**【回想: 1970年代、ニューヨーク】**

**場所**: ウォール街、投資顧問会社のオフィス

**アペル**: 「移動平均線...便利だが、遅すぎる」

*チャートに引かれた移動平均線を見つめる。*

**アペル**: 「価格が動いてから、移動平均線が反応するまでタイムラグがある」

**アペル**: 「もっと早く、トレンドの転換を捉えられないか？」

---

**【物理学者の着想】**

*アペルは机の引き出しから、大学時代の物理学の教科書を取り出す。*

**アペル**: 「待てよ...物理学で考えたらどうだ？」

*ページをめくり、運動方程式の章を開く。*

```
物理学の基本:

位置 x(t)
速度 v(t) = dx/dt （位置の変化率）
加速度 a(t) = dv/dt （速度の変化率）

加速度 = どれだけ速く加速しているか
```

**アペル**: 「そうか...価格を『位置』と考えたら...」

*紙に書き始める。*

```
価格の物理学的解釈:

価格 = 位置
移動平均線 = 速度（価格の平均的な動き）

短期MA(12日) = 短期的な速度
長期MA(26日) = 長期的な速度

差 = 短期速度 - 長期速度 = ？
```

**アペル**: 「...これは**加速度**だ！」

---

**【発見】**

```
MACD = 短期MA - 長期MA
     = 短期速度 - 長期速度
     = 加速度（モメンタムの変化）

意味:
MACDがプラス → 短期速度 > 長期速度
              → 価格が加速している

MACDがマイナス → 短期速度 < 長期速度
                → 価格が減速している

MACDがゼロ → 短期速度 = 長期速度
            → 等速運動（トレンド転換点）
```

**アペル**: 「移動平均線は『速度』を見ていた。でも俺は『加速度』を見れば、もっと早く転換点を捉えられる！」

---

**【しかし、問題が発生】**

*翌日、アペルは過去のチャートにMACDを引いてみる。*

**アペル**: 「...使えない」

```
MACD線:
     ●
  ●   ●  ●
●  ●   ●   ● ← 毎日上下にブレる（ノイズ）

問題:
- いつ買えばいいか分からない
- 上がったり下がったりで判断できない
```

**アペル**: 「加速度は分かった。でも、ノイズが多すぎて使えない...」

*机の上に、昔の同僚から送られてきた制御工学の論文がある。手に取る。*

**アペル**: 「待てよ...制御工学では、ノイズをどう処理してた？」

---

**【制御工学のフィルター理論】**

```
制御工学の基本:

センサーデータ → ノイズが多い
        ↓
   移動平均フィルター
        ↓
  滑らかなデータ

例: 飛行機の高度計
生データ: 100m, 102m, 99m, 103m, 100m...（ブレる）
フィルター後: 100m, 100.5m, 101m...（滑らか）
```

**アペル**: 「そうか！MACD線にも『フィルター』をかければいい！」

*紙に書き始める。*

```
発想:
MACD線 = 生の加速度（ノイズあり）
     ↓
  移動平均（9日）
     ↓
Signal線 = 滑らかな加速度（ノイズ除去）

2本の線を比較:
MACD線 > Signal線 → 加速度が平均より大きい
                  → 加速が強まった瞬間！

MACD線 < Signal線 → 加速度が平均より小さい
                  → 減速が始まった瞬間！
```

**アペル**: 「これだ！ダブルチェックで『今が買い時』を明確にできる！」

---

**【検証】**

*過去10年分のチャートに、MACD線とSignal線を引く。*

**アペル**: 「...完璧だ」

```
検証結果:
- ゴールデンクロス（MACD > Signal）の時に買う
  → 70%以上が利益

- デッドクロス（MACD < Signal）の時に売る
  → 損失を最小化
```

**アペル**: 「これだ！Moving Average Convergence Divergence...MACD！」

*1979年、アペルはこの技術をニュースレターで発表した。*

---

**【現代に戻る】**

**ミコ**: 「これがMACDの誕生だ」

**ユウタ**: 「加速度...？難しそう」

### Mission 3: MACDを理解する

**ミコ**: 「簡単だ。物理学で考えろ」

```
物理学的解釈:

価格 = 位置（どこにいるか）
移動平均線 = 速度（どれだけ速く動いているか）
MACD = 加速度（どれだけ速く加速しているか）

例:
車の運転で考える:

速度メーター: 時速60km → 移動平均線
アクセル踏み込み: 加速中 → MACDプラス
ブレーキ踏み込み: 減速中 → MACDマイナス
```

**ユウタ**: 「なるほど！加速してるか減速してるかを見るんだ」

**ミコ**: 「その通り。でも、アペルはもう一つ問題に直面した」

**ユウタ**: 「問題？」

**ミコ**: 「加速度は分かったが、**ノイズが多すぎた**」

```
MACD線だけだと:
月曜: +0.05
火曜: +0.03
水曜: +0.06
木曜: +0.02
金曜: +0.07

→ 毎日上下してて、いつ買えばいいか分からない
```

**ユウタ**: 「確かに...これじゃ使えないな」

**ミコ**: 「だから、アペルは**フィルター**をかけた」

**ユウタ**: 「フィルター？」

**ミコ**: 「制御工学の技術だ。ノイズを除去して、滑らかにする」

```
MACD線: 短期MA(12日) - 長期MA(26日)
       = 生の加速度（ノイズあり）

Signal線: MACD線の移動平均（9日）
        = 滑らかな加速度（ノイズ除去後）
```

**ユウタ**: 「ノイズを除去...それで2本なんだ」

**ミコ**: 「そう。この2本の線の**交差**が、トレンド転換のサインだ」

**ミコ**: 「これを**ダブルチェック**と呼ぶ」

```
ダブルチェックの意味:

【シングルチェック（MACD線だけ）】
MACD > 0 → 「一応、上昇トレンド」
→ でも、ノイズで毎日変わる（曖昧）

【ダブルチェック（Signal線を追加）】
MACD > Signal → 「加速が強まった瞬間！」
→ 明確な買いシグナル
```

**ユウタ**: 「2段階で確認するから、信頼性が上がるんだ！」

**ミコ**: 「完璧だ」

```
【ゴールデンクロス】（買いシグナル）

Signal線
    ╲
     ╲    ╱← MACD線
      ╲  ╱
       ╲╱
       ╱╲  ← 交差（ゴールデンクロス）
      ╱  ╲
     ╱    ╲

意味: MACD線がSignal線を下から上に抜ける
→ 上昇トレンド開始
```

**ユウタ**: 「下から上に抜けたら、買い？」

**ミコ**: 「その通り」

```
【デッドクロス】（売りシグナル）

MACD線
    ╲
     ╲    ╱← Signal線
      ╲  ╱
       ╲╱
       ╱╲  ← 交差（デッドクロス）
      ╱  ╲
     ╱    ╲

意味: MACD線がSignal線を上から下に抜ける
→ 下降トレンド開始
```

**ユウタ**: 「上から下に抜けたら、売り」

**ミコ**: 「完璧だ」

---

**【Histogram（棒グラフ）】**

**ミコ**: 「さらに、MACDには**Histogram**がある」

```
Histogram = MACD線 - Signal線の差
          = 加速度 - 加速度の平均
          = 加速度の「ブレ」

     ┃
  ▓▓▓┃     ← プラス（加速度が平均より大きい）
 ▓▓▓▓┃        → 上昇トレンド加速中
━━━━━┃━━━━━ ← ゼロライン（加速度 = 平均）
     ┃▓▓▓    ← マイナス（加速度が平均より小さい）
     ┃▓▓▓▓      → 下降トレンド加速中
```

**ユウタ**: 「棒の高さで、加速の強さが分かる？」

**ミコ**: 「その通り。棒が長い = 加速が強い = トレンド強い」

```
物理学的に:
Histogram拡大 → 加速度が増加 → アクセル踏み込み中
Histogram縮小 → 加速度が減少 → ブレーキ踏み始め
Histogramゼロ → 加速度ゼロ → 等速運動（転換点）
```

**ユウタ**: 「なるほど、視覚的に分かりやすい！車の運転と同じだ」

## Part 5: ボリンジャーバンド - 値動きの範囲を予測（45分）

### Scene 7: ジョン・ボリンジャーの発見

**ミコ**: 「最後の武器は、ボリンジャーバンドだ」

**ユウタ**: 「ボリンジャー...？」

**ミコ**: 「価格の『正常範囲』を視覚化する技術だ。1980年代、ジョン・ボリンジャーが発明した」

---

**【回想: 1980年代、カリフォルニア】**

**場所**: ボリンジャーの自宅、深夜

**ボリンジャー**: 「移動平均線だけでは足りない...価格がどこまで動くか予測できない」

*過去数年分のチャートを並べている。*

**ボリンジャー**: 「待てよ...大学で統計学を学んだことを思い出せ」

*本棚から統計学の教科書を取り出す。正規分布の章を開く。*

```
統計学の基本: 正規分布

平均値を中心に、データは散らばる

標準偏差σ（シグマ）:
±1σ: 約68%のデータが収まる
±2σ: 約95%のデータが収まる
±3σ: 約99.7%のデータが収まる
```

**ボリンジャー**: 「もし価格も正規分布に従うなら...」

*紙に書き出す。*

```
仮説:
価格は移動平均線を中心に、正規分布する

移動平均線 ± 2σ = 価格の95%が収まる範囲

つまり:
上限 = 移動平均線 + 2σ
下限 = 移動平均線 - 2σ

この範囲を超える = 異常事態（5%の確率）
```

**ボリンジャー**: 「検証してみよう...」

*過去10年分のチャートに、移動平均線±2σを引いていく。*

**ボリンジャー**: 「...信じられない。95%以上の価格が、この範囲内に収まってる！」

**ボリンジャー**: 「これだ！統計学で価格の『正常範囲』が可視化できる！」

*1980年代、ボリンジャーはこの技術を「ボリンジャーバンド」と名付けた。*

---

**【現代に戻る】**

**ミコ**: 「ボリンジャーは統計学で、価格の『正常範囲』を発見した」

**ユウタ**: 「95%...？」

**ミコ**: 「それが標準偏差±2σ（シグマ）だ。難しく聞こえるが、使うのは簡単」

### Mission 4: ボリンジャーバンドを理解する

**ミコ**: 「ボリンジャーバンドは3本の線で構成される」

```
ボリンジャーバンド:

━━━━━━━━━━━━━━ ← 上限（移動平均 + 2σ）
                    95%の範囲の上限
～～～～～～～～～～～～ ← 中央線（移動平均線）
                    価格の中心
━━━━━━━━━━━━━━ ← 下限（移動平均 - 2σ）
                    95%の範囲の下限

統計学的意味:
価格の95%は、上限と下限の間に収まる
残り5%は範囲外 = 異常事態
```

**ユウタ**: 「上限・下限を超えたら...？」

**ミコ**: 「異常事態だ。100回中5回しか起きない」

**ミコ**: 「つまり、**そろそろ正常範囲に戻る確率が高い**」

```
統計学的解釈:

価格が上限に触れる:
→ 95%の範囲を超えている（上位2.5%）
→ 確率的に異常 → 下がる可能性が高い

価格が下限に触れる:
→ 95%の範囲を下回っている（下位2.5%）
→ 確率的に異常 → 上がる可能性が高い
```

**ユウタ**: 「確率で判断するんだ！」

**ミコ**: 「そう。統計学は『確率』の学問だ」

```
追加の使い方:
- バンドが狭まる（スクイーズ）
  → 標準偏差が小さい
  → 価格変動が少ない
  → 大きな動きの前触れ（エネルギー蓄積中）

- バンドが広がる（エクスパンション）
  → 標準偏差が大きい
  → 価格変動が激しい
  → トレンド発生中
```

**ユウタ**: 「統計学で市場を読むのか。カッコいい」

**ミコ**: 「これで5つの武器が揃った。最後に、一番重要なことを教える」

## Part 6: リスク管理 - 生き残るための鉄則（30分）

### Scene 8: ミコの失敗談

**ユウタ**: 「一番重要なこと...？」

**ミコ**: 「**リスク管理**だ」

**ユウタ**: 「リスク管理...？」

**ミコ**: 「技術を学んでも、リスク管理ができないと負ける。俺がそうだった」

**ユウタ**: 「ミコも負けたの？」

**ミコ**: 「Scene 3で話した失敗談を覚えてるか？5万円が500円になった話」

**ユウタ**: 「覚えてる」

**ミコ**: 「あれは、リスク管理をしなかったから負けた」

```
ミコの失敗:
- 損切りしなかった → -50%でも「戻るかも」と塩漬け
- 全資金を一つに集中 → 一つ失敗で全滅
- 感情で判断 → 「みんな買ってるから」で買った

結果: 5万円 → 500円
```

**ミコ**: 「技術があっても、ルールを守らないと意味がない」

**ユウタ**: 「...怖いな」

**ミコ**: 「だから、最初にルールを決める」

### Mission 5: ルールを決める

**ミコ**: 「お前のルールを決めよう」

```
【4つの鉄則】

1. 損切りライン
   購入価格の -10%
   → ここを割ったら、機械的に売る

2. 利確ライン
   購入価格の +20%
   → ここに達したら、機械的に売る

3. 分散投資
   1銘柄は総資金の20%まで
   → 一つ失敗しても致命傷にならない

4. 感情のコントロール
   FOMO（取り残される恐怖）に負けない
   → 「みんな買ってる」は買う理由にならない
```

**ユウタ**: 「機械的に...？」

**ミコ**: 「そう。感情を入れるな。-10%になったら、理由を考えずに売る」

**ミコ**: 「『もうちょっと待てば戻るかも』は禁句だ」

**ユウタ**: 「厳しいな...」

**ミコ**: 「厳しくないと生き残れない」

**ミコ**: 「勝つことより、**負けないこと**が先だ」
