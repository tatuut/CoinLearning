# Week 1: 基礎知識と最初の取引

## 🎯 目標
- 草コインとは何かを理解する
- 取引所に登録して入金する
- 最初の100円で実験取引を行う

## 📚 学ぶこと

### 1. 草コインとは？
- **定義**: 時価総額が低く、知名度の低い仮想通貨
- **特徴**:
  - 価格が安い（1コイン=0.0001円とか）
  - ボラティリティ（価格変動）が超激しい
  - 1日で10倍になることもあれば、ゼロになることも
- **リスク**: ⚠️ 投資した金額が全部なくなる可能性がある

### 2. 取引所の選び方

#### おすすめ：Binance（バイナンス）
- **メリット**:
  - 草コイン数百種類
  - 取引量が多い（流動性が高い）
  - API対応で自動化可能
  - 新規登録キャンペーン多数

- **デメリット**:
  - 日本語対応が限定的
  - 最低入金額が少し高い（$10-15程度）

#### 代替案：MEXC
- 超マイナー草コイン多数
- 最低取引額が低い（$5程度）

### 3. 取引所登録の手順

1. **メールアドレスで登録**
2. **本人確認（KYC）** - パスポートか免許証
3. **二段階認証を設定** - セキュリティ重要！
4. **入金** -
   - 日本の取引所でXRP/USDTを買う
   - 海外取引所に送金（手数料安い）
   - 草コインを買える状態に

### 4. 最初の100円取引

#### ステップ：
```
1. ウォッチリストから草コインを選ぶ
   → 出来高が多いものを選ぶ（流動性重要）

2. 少額（100円分）で買い注文
   → 成行注文で即座に買う

3. 値動きを観察
   → なぜ上がった？なぜ下がった？

4. 売却タイミングを決める
   → +10%で利確？ -5%で損切り？

5. 結果を記録
   → trades.db に自動保存される
```
