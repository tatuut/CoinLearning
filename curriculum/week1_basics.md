# Week 1: ユウタの冒険が始まる

## 登場人物

**ユウタ（23歳）** - あなたと一緒に100円から1000円を目指す主人公
バイト代が少なくて悩んでいたところ、「草コイン」という言葉を知る。
「100円から始められるなら...」そう思って、この冒険を始めることにした。

**ミコ** - 謎のAIトレーダー
過去に何度も失敗してきたけど、今は安定して稼げるようになった。
ユウタのメンターとして、失敗も含めて全部教えてくれる。

---

## 📖 Story: Day 1 - 運命の出会い

### ユウタの独白

「はぁ...今月もバイト代ギリギリだな...」

スマホを見ながらため息をつくユウタ。バイト代は月8万円。家賃5万、食費2万、光熱費1万。残りは雀の涙だ。

**「海外旅行とか行きたいのに、金がないんだよな...」**

そんな時、SNSで「100円が1000円になった！」という投稿を見つけた。

「草コイン...？仮想通貨か。でも100円なら...失っても痛くないし、やってみるか」

---

### ミコ登場

翌日、ユウタがネットで調べていると、突然画面に謎のキャラクターが現れた。

**ミコ**: 「よお、新人。草コイン始めるんだって？」

**ユウタ**: 「えっ、誰!? なんかAIっぽいキャラが...」

**ミコ**: 「私はミコ。AIトレーダーだよ。あんたみたいな初心者を見ると、放っておけなくてさ」

**ユウタ**: 「マジで!? じゃあ、教えてくれるの？」

**ミコ**: 「ああ。でも最初に言っておく。**失敗する**よ、絶対に」

**ユウタ**: 「え...」

**ミコ**: 「誰だって最初は失敗する。私も最初の取引で50%損したからね。でも大丈夫。100円なら学習コストとして安いもんだ」

**ユウタ**: 「...わかった。一緒に1000円目指そう！」

---

## 📚 ミコの教え: 草コインとは何か

### ミコの説明

**ミコ**: 「まず基本から。草コインって何か知ってる？」

**ユウタ**: 「えーと...安いコイン？」

**ミコ**: 「そう！正確には『時価総額が低くて、誰も知らないような仮想通貨』のこと」

#### 草コインの特徴

```
【メリット】
✅ 1日で10倍、100倍になることもある
✅ 少額（100円）から始められる
✅ 当たればデカい

【デメリット】
❌ 1日で価値がゼロになることもある
❌ 詐欺プロジェクトも多い
❌ めちゃくちゃリスクが高い
```

**ユウタ**: 「うわ...ギャンブルじゃん...」

**ミコ**: 「ギャンブルじゃない。**学習と戦略だよ**」

**ミコ**: 「私が最初に負けたのは、何も知らずに『みんなが買ってるから』って理由で買ったから。でも今は違う。ちゃんと分析して、ルール守って、記録つけて...そうしたら勝てるようになった」

**ユウタ**: 「じゃあ、俺も勝てる？」

**ミコ**: 「勝てる。でも**最初は負ける**。それを前提に行こう」

---

## 🎯 Mission 1: 取引所に登録しよう

### ミコのアドバイス

**ミコ**: 「じゃあ、最初のミッション。取引所に登録だ」

**ユウタ**: 「取引所？」

**ミコ**: 「コインを買う場所だよ。おすすめは**Binance（バイナンス）**。理由は...」

#### Binanceを選ぶ理由

1. **草コインが超豊富** - 数百種類以上
2. **手数料が安い** - 0.1%程度
3. **API対応** - 自動化できる（後で使う）
4. **新規キャンペーンが多い** - 手数料割引とかある

**ユウタ**: 「でも海外の取引所でしょ？大丈夫なの？」

**ミコ**: 「世界最大級だから安全性は高い。ただし、日本の銀行から直接入金はできないから、ちょっとコツがいる」

### 登録手順

```
1. Binanceのサイトにアクセス
2. メールアドレスで登録
3. 本人確認（KYC）
   → パスポートか免許証が必要
4. 二段階認証を設定
   → これ超重要！セキュリティのため！
```

**ユウタ**: 「登録できた！」

**ミコ**: 「よし、次は入金だ」

---

## 💰 Mission 2: 初めての入金（これが最初の難関）

### ミコの警告

**ミコ**: 「ここが最初の難関。海外取引所への入金は、ちょっと面倒なんだ」

**ユウタ**: 「なんで？」

**ミコ**: 「日本の銀行から海外取引所には直接送金できない。だから、日本の取引所を経由する必要がある」

#### 入金の流れ

```
1. 日本の取引所（bitFlyer/Coincheckなど）でXRPかUSDTを購入
   → 手数料が安いコインを選ぶ

2. 購入したコインをBinanceに送金
   → 送金手数料が安い（XRPなら数円）

3. Binanceで草コインを買える状態に！
```

**ユウタ**: 「めんどくせぇ...」

**ミコ**: 「最初だけだから我慢！慣れたら5分で終わるよ」

---

**ユウタ**: 「よし、入金完了！で、何買えばいいの？」

**ミコ**: 「待て待て！適当に買うなよ」

**ミコ**: 「最初は**出来高が多いコイン**を選べ。流動性が低いと、売りたい時に売れないからな」

**ミコ**: 「理由は？」


**ミコ**: 「悪くない。出来高も多いし、上昇トレンドっぽい。じゃあ100円分買ってみろ」

## 😱 Story: Day 3 - 最初の失敗

### 3時間後...

**ユウタ**: 「ミコ！見て！+8%になってる！」

```
SHIB/USDT
現在価格: $0.00001134
損益: +8.0% (+8円)
```

**ユウタ**: 「このまま行けば、すぐ1000円いけるんじゃね!?」

**ミコ**: 「待て。そういう時こそ冷静になれ」

**ミコ**: 「利確（利益確定）するか、それとも持ち続けるか。ルールを決めておくべきだった」

**ユウタ**: 「えー、でもまだ上がりそうだし...」

---

### 翌日...

**ユウタ**: 「あああああ！-12%になってる！！！」

```
SHIB/USDT
現在価格: $0.00000924
損益: -12.0% (-12円)
```

**ミコ**: 「...な？」

**ユウタ**: 「なんで!? 昨日は+8%だったのに！」

**ミコ**: 「それが草コインだ。1日で±20%なんて普通」

---

### ミコの教訓

**ミコ**: 「ユウタ、今回の失敗で学んだことは？」

**ユウタ**: 「...利益が出てる時に売っておけばよかった」

**ミコ**: 「そう。これが**利確**の重要性だ」

**ミコ**: 「もう一つ。**損切り**も重要。-10%になったら諦めて売る、みたいなルールを決めておくべきだった」

#### 取引のルール（ミコ版）

```
【ミコの鉄則】

1. 利確ライン: +20-30%
   → このラインに到達したら売る

2. 損切りライン: -10%
   → これ以上は損失を拡大させない

3. ルールを守る
   → 感情で判断しない
```

**ユウタ**: 「わかった...次はルール守る」

**ミコ**: 「よし。じゃあもう一度、今度はルール付きでやってみろ」
