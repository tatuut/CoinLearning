## ステップ1: Binance（バイナンス）に登録する

### 1-1. アカウント作成

1. Binanceの公式サイトにアクセス
   - URL: https://www.binance.com/ja
   - ⚠️ 注意: フィッシングサイトに注意！必ず公式URLを確認

2. 「登録」ボタンをクリック

3. メールアドレスまたは電話番号を入力
   - メールアドレス推奨（管理しやすい）
   - パスワードを設定（英数字+記号で強力なものに）

4. 認証コードを入力
   - メールに届いた6桁のコードを入力

5. 利用規約に同意してアカウント作成完了

### 1-2. 本人確認（KYC）を完了する

取引するには本人確認が必須です。

1. ダッシュボードから「本人確認」セクションに移動

2. 必要な情報を入力
   - 氏名（パスポート/免許証と同じ）
   - 生年月日
   - 住所
   - 国籍

3. 本人確認書類をアップロード
   - パスポート（推奨）
   - 運転免許証
   - マイナンバーカード

4. セルフィー撮影（自撮り）
   - 指示に従って顔写真を撮影

5. 審査を待つ（通常1〜3日）
   - 早ければ数時間で完了

### 1-3. 二段階認証（2FA）を設定する【超重要】

セキュリティのため必ず設定してください。

1. 「セキュリティ」→「二段階認証」に移動

2. Google Authenticatorをスマホにインストール
   - iOS: App Store
   - Android: Google Play

3. QRコードをスキャン

4. 6桁の認証コードを入力して設定完了

⚠️ **バックアップキーを必ず保存**
   - スマホを紛失した時に必要
   - 紙に書いて安全な場所に保管

## ステップ2: 日本円を仮想通貨に変えてBinanceに送金する

Binanceには日本円を直接入金できません。日本の取引所を経由します。

### 2-1. 日本の取引所でXRPまたはUSDTを購入

#### おすすめの日本の取引所

- **bitFlyer**（ビットフライヤー）
- **Coincheck**（コインチェック）
- **GMOコイン**

どれか1つに登録してください（すでに登録済みならそれを使う）。

#### 手順（例: bitFlyerの場合）

1. bitFlyerにログイン

2. 日本円を入金
   - 「入出金」→「日本円」→「銀行振込」
   - 最低500円〜1000円を入金（余裕を持たせる）
   - 振込手数料に注意

3. XRP（リップル）を購入
   - なぜXRP？→ 送金手数料が安い（数円）
   - 「販売所」ではなく「取引所」で買う（手数料が安い）
   - 500円分のXRPを購入

### 2-2. XRPをBinanceに送金する

1. Binanceで入金アドレスを取得
   - Binanceにログイン
   - 「ウォレット」→「フィアットと現物」
   - 「入金」をクリック
   - 通貨で「XRP」を選択
   - **アドレス**と**メモ/タグ**をコピー

   ⚠️ **超重要**: XRPの送金には「アドレス」と「メモ（タグ）」の両方が必要！
   どちらか片方でも間違えると資金が消えます

2. bitFlyerから送金
   - 「入出金」→「XRP」→「送付」
   - Binanceのアドレスを入力
   - **メモ（タグ）も必ず入力**
   - 送金額を入力（500円分のXRP）
   - 確認して送金

3. 着金を待つ
   - 通常5〜15分で着金
   - Binanceの「ウォレット」で確認

### 2-3. XRPをUSDTに交換する

草コインを買うにはUSDT（テザー）が必要です。

1. Binanceで「トレード」→「現物」を選択

2. 「XRP/USDT」を検索

3. XRPを全額売却してUSDTにする
   - 「売却」タブ
   - 「100%」をクリック（全額売却）
   - 「売却」ボタン

4. USDTの残高を確認
   - これで草コインを買う準備完了！

```
選択: 2
```

### 5-2. シンボルを入力

例: PEPE（ペペコイン）を分析したい場合

```
シンボルを入力（例: SHIBUSDT）: PEPEUSDT
```

⚠️ **重要**: シンボルは必ず「USDT」で終わる形式で入力
   - PEPEUSDT ⭕
   - PEPE ❌

### 5-3. 分析結果を確認

システムが3つの戦略で分析した結果を表示します：

```
[*] PEPEUSDT を全戦略で分析中...

【モメンタム戦略】
  [OK] モメンタム上昇トレンド
  価格: $0.00000120
  モメンタム: 8.45%

【出来高急増戦略】
  [OK] 出来高急増＋上昇
  価格: $0.00000120
  出来高: 4.2x

【ブレイクアウト戦略】
  [NG] 条件不足

------------------------------------------------------------
【総合判定】
[STRONG] 強い買いシグナル！複数の戦略が一致しています。
```

### 5-4. 総合判定を見て判断する

- **[STRONG]**: 複数の戦略が一致 → 買いのチャンス
- **[OK]**: 1つの戦略のみ → 慎重に判断
- **[NG]**: シグナルなし → 見送り

## ステップ6: 実際に取引する（Binanceで）

### 6-1. Binanceにログイン

分析結果をもとに、実際にBinanceで取引します。

### 6-2. コインを検索

1. 「トレード」→「現物」を選択
2. 検索ボックスに「PEPE/USDT」を入力

### 6-3. 購入する

1. 「買い」タブを選択
2. 価格：「成行」を選択（すぐに買える）
3. 金額：100円分（約1.5 USDT）を入力
4. 「購入」ボタンをクリック

### 6-4. ルールを設定する

**超重要**: 買った瞬間にルールを決める！

```
【あなたのルール】
利確ライン: +25%
損切りライン: -10%
```

このラインに達したら必ず売ります（感情に流されない）。

## ❓ よくある質問（Week 1編）

### Q1: Binanceの本人確認が通らない

A: 以下を確認してください：
- 書類の写真が鮮明か
- 入力した情報と書類が一致しているか
- セルフィーで顔がはっきり写っているか

### Q2: シグナルが全然出ない

A: 正常です。市場が下降トレンドの時はシグナルが出にくいです。
   - 別の戦略を試す
   - 時間帯を変える（日本時間の夜〜深夜が活発）
   - 焦らず待つ

### Q3: 100円じゃ少なすぎる？

A: Week 1は学習期間なので100円で十分です。
   慣れてきたら200円、300円と増やしていきましょう。

### Q4: 損切りが怖くてできない

A: その気持ちはわかります。でも損切りしないと：
   - -50%、-80%と傷が広がる
   - 次の取引機会を逃す

   「-10%は授業料」と割り切りましょう。

### Q5: システムが動かない

A: 以下を確認：
   ```bash
   python --version  # Python 3.8以上か？
   pip list  # requests, numpy がインストール済みか？
   ```
