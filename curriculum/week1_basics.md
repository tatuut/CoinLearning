# Week 1: ユウタの冒険が始まる

## 登場人物

**ユウタ（23歳）** - あなたと一緒に100円から1000円を目指す主人公
バイト代が少なくて悩んでいたところ、「草コイン」という言葉を知る。
「100円から始められるなら...」そう思って、この冒険を始めることにした。

**ミコ** - 謎のAIトレーダー
過去に何度も失敗してきたけど、今は安定して稼げるようになった。
ユウタのメンターとして、失敗も含めて全部教えてくれる。

---

「はぁ...今月もバイト代ギリギリだな...」

**「海外旅行とか行きたいのに、金がないんだよな...」**

「草コイン...？仮想通貨か。でも100円なら...失っても痛くないし、やってみるか」

## 📚 ミコの教え: 草コインとは何か

### ミコの説明

**ミコ**: 「まず基本から。草コインって何か知ってる？」

**ユウタ**: 「えーと...安いコイン？」

**ミコ**: 「そう！正確には『時価総額が低くて、誰も知らないような仮想通貨』のこと」

#### 草コインの特徴

```
【メリット】
✅ 1日で10倍、100倍になることもある
✅ 少額（100円）から始められる
✅ 当たればデカい

【デメリット】
❌ 1日で価値がゼロになることもある
❌ 詐欺プロジェクトも多い
❌ めちゃくちゃリスクが高い
```

**ユウタ**: 「うわ...ギャンブルじゃん...」

**ミコ**: 「ギャンブルじゃない。**学習と戦略だよ**」

**ミコ**: 「私が最初に負けたのは、何も知らずに『みんなが買ってるから』って理由で買ったから。でも今は違う。ちゃんと分析して、ルール守って、記録つけて...そうしたら勝てるようになった」

**ユウタ**: 「じゃあ、俺も勝てる？」

**ミコ**: 「勝てる。でも**最初は負ける**。それを前提に行こう」

## 🎯 Mission 1: 取引所に登録しよう

### ミコのアドバイス

**ミコ**: 「じゃあ、最初のミッション。取引所に登録だ」

**ユウタ**: 「取引所？」

**ミコ**: 「コインを買う場所だよ。おすすめは**Binance（バイナンス）**。理由は...」

#### Binanceを選ぶ理由

1. **草コインが超豊富** - 数百種類以上
2. **手数料が安い** - 0.1%程度
3. **API対応** - 自動化できる（後で使う）
4. **新規キャンペーンが多い** - 手数料割引とかある

**ユウタ**: 「でも海外の取引所でしょ？大丈夫なの？」

**ミコ**: 「世界最大級だから安全性は高い。ただし、日本の銀行から直接入金はできないから、ちょっとコツがいる」

### 登録手順

```
1. Binanceのサイトにアクセス
2. メールアドレスで登録
3. 本人確認（KYC）
   → パスポートか免許証が必要
4. 二段階認証を設定
   → これ超重要！セキュリティのため！
```

**ユウタ**: 「登録できた！」

**ミコ**: 「よし、次は入金だ」

# 📋 実際の手順 - Week 1を実践する

ここからは、あなたが実際にユウタと同じ経験をするための**具体的な手順**を説明します。

## 🎯 目標

- Binanceアカウントを開設する
- 100円〜500円を入金する
- 草コイントレーダーシステムを使えるようにする
- 最初の取引を記録する

---

## ステップ1: Binance（バイナンス）に登録する

⏱️ **所要時間**: 約10〜15分

### 1-1. アカウント作成（5分）

#### 手順1: Binanceのサイトを開く

1. **ブラウザを開く**
   - Chrome、Edge、Firefoxなど、普段使っているブラウザでOK

2. **アドレスバーに以下を入力**
   ```
   https://www.binance.com/ja
   ```

   ⚠️ **超重要**: 必ず上記のURLを直接入力してください
   - Google検索からアクセスしない（偽サイトの危険）
   - メールのリンクをクリックしない
   - URLが `binance.com` であることを確認

3. **画面が表示される**
   - 黄色と黒を基調とした画面が表示されます
   - 右上に「ログイン」「登録」ボタンがあります

#### 手順2: 登録ボタンをクリック

1. **画面右上を見る**
   - 右上に「ログイン」「登録」という2つのボタンが並んでいます

2. **「登録」ボタンをクリック**
   - 黄色いボタンです
   - クリックすると新しい画面が表示されます

#### 手順3: メールアドレスを入力

新しい画面が表示されます。以下の項目があります：

1. **「個人アカウントの作成」という見出しがあることを確認**

2. **「メール」「電話番号」の選択タブがある**
   - 「メール」タブが選ばれていることを確認
   - 電話番号でも登録できますが、メール推奨

3. **メールアドレスを入力**
   - 入力欄: 「メールアドレス」
   - あなたが普段使っているGmail、Yahoo等のメールを入力
   - 例: `taro.yamada@gmail.com`

4. **パスワードを作成**
   - 入力欄: 「パスワード」
   - 条件: 8文字以上、大文字・小文字・数字・記号を含む
   - 例: `MyPassword123!`
   - ⚠️ 必ずメモしてください（忘れると困ります）

5. **パスワードを再入力**
   - 入力欄: 「パスワードの確認」
   - 上で入力したのと同じパスワードをもう一度入力

6. **紹介ID（任意）**
   - 空欄でOK（スキップ）

7. **利用規約にチェック**
   - 「私は18歳以上で、Binanceの利用規約を読み、同意します」
   - この左側にある小さな四角いチェックボックスをクリック
   - チェックマークが入ります

8. **「アカウント作成」ボタンをクリック**
   - 黄色い大きなボタンです
   - 画面下部にあります

#### 手順4: パズルを解く（セキュリティチェック）

1. **スライダーパズルが表示される**
   - 画面に絵のパズルが表示されます
   - 「スライダーをドラッグしてパズルを完成させてください」

2. **スライダーを動かす**
   - 画面下部に横長のスライダーがあります
   - マウスでスライダーをクリックして**右にドラッグ**
   - パズルのピースが動きます

3. **ピースが正しい位置に来るまでドラッグ**
   - ピースがぴったり合う場所で離します
   - 正解すると次の画面に進みます
   - 失敗したらもう一度トライ

#### 手順5: メール認証コードを入力

1. **「メール認証」画面が表示される**
   - 「メールアドレスに認証コードを送信しました」と表示

2. **メールを確認**
   - 別のタブまたはウィンドウで、あなたのメールを開く
   - Gmail、Yahoo等、登録したメールアドレスのメールボックス

3. **Binanceからのメールを探す**
   - 件名: 「Binanceアカウント登録の認証」
   - 送信者: `do_not_reply@directmail.binance.com`
   - ⏱️ 数秒〜1分で届きます（届かない場合は迷惑メールフォルダをチェック）

4. **メールを開いて6桁のコードをコピー**
   - メール本文に6桁の数字があります
   - 例: `123456`
   - この数字を選択してコピー（右クリック→コピー、またはCtrl+C）

5. **Binanceの画面に戻る**
   - ブラウザのタブを切り替え

6. **6桁のコードを入力**
   - 6つの入力欄があります
   - コピーしたコードを貼り付け（Ctrl+V）、または手動で入力
   - 自動的に次の画面に進みます

#### 手順6: アカウント作成完了

✅ **おめでとうございます！アカウントが作成されました**

- 画面に「アカウント作成完了」または「ウェルカム」と表示されます
- これでBinanceにログインできる状態になりました

**重要**: 次は本人確認（KYC）が必要です。このまま続けましょう。

### 1-2. 本人確認（KYC）を完了する

⏱️ **所要時間**: 約10分（審査に1〜3日）

⚠️ **重要**: 取引するには本人確認が必須です。この作業を完了しないと、コインを買えません。

#### 準備するもの

以下のいずれか1つを手元に用意してください：

- ✅ **パスポート**（推奨 - 審査が早い）
- ✅ **運転免許証**
- ✅ **マイナンバーカード**

スマホを使うと簡単です（書類とセルフィーを撮影）。

---

#### 手順1: 本人確認ページに移動

1. **Binanceのトップページを表示**
   - ログインした状態で、画面右上の**人型アイコン**をクリック
   - またはURLバーに `https://www.binance.com/ja` を入力

2. **メニューが表示される**
   - 画面右上の人型アイコンをクリックすると、ドロップダウンメニューが開きます

3. **「本人確認」を選択**
   - メニューの中から「本人確認」または「Identity Verification」をクリック
   - 黄色いバッジのようなアイコンと一緒に表示されています

4. **本人確認ページが開く**
   - 「本人確認」という大きな見出しのページが表示されます
   - 下に「認証レベル」という説明があります

#### 手順2: 認証レベルを選択

1. **「Verified」または「Verified Plus」を選択**
   - 画面に2つまたは3つの選択肢があります
   - **「Verified」を選択**（通常はこれで十分）
   - 黄色い「今すぐ確認」ボタンをクリック

#### 手順3: 国と書類の種類を選択

1. **発行国を選択**
   - ドロップダウンメニューから「日本」を選択
   - 「Japan」または「日本」と表示されます

2. **書類の種類を選択**
   - 以下から1つ選びます：
     - **パスポート**（Passport）- 推奨
     - **運転免許証**（Driver's License）
     - **マイナンバーカード**（ID Card）

3. **「続行」ボタンをクリック**

#### 手順4: 個人情報を入力

画面に入力フォームが表示されます。以下を入力してください：

1. **氏名（First Name / Last Name）**
   - ⚠️ **超重要**: 書類と完全に一致させる
   - 例:
     - First Name（名）: `Taro`
     - Last Name（姓）: `Yamada`
   - ローマ字で入力（パスポートと同じ表記）

```
4. 次に改善すること
   → （メモ）
```

   - -50%、-80%と傷が広がる
   - 次の取引機会を逃す

   「-10%は授業料」と割り切りましょう。
