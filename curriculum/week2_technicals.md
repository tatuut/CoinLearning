## 📚 ミコの特別講義: テクニカル分析とは

### 講義室（脳内）

**ミコ**: 「テクニカル分析って知ってるか？」

**ユウタ**: 「なんか...チャート見るやつ？」

**ミコ**: 「そう。価格の動きを分析して、『今が買い時か、売り時か』を判断する技術だ」

#### なぜテクニカル分析が必要？

```
【運だけの取引】
「なんとなく上がりそう」
「みんな買ってるから」
→ 根拠がない
→ いつ売るか分からない
→ 感情で判断してしまう

【テクニカル分析】
「RSIが30以下だから買い時」
「MACDがゴールデンクロスしたから買い」
→ 明確な根拠がある
→ ルールに基づいて判断できる
→ 感情に左右されない
```

**ユウタ**: 「なるほど...でも難しそう...」

**ミコ**: 「最初は難しく感じるけど、慣れれば簡単。今週は3つの指標を教える」

## 🎓 Lesson 1: ローソク足 - チャートの言語を学ぶ

### ミコの説明

**ミコ**: 「まず、ローソク足から。これが読めないと話にならない」

**ユウタ**: 「ローソク？蝋燭？」

**ミコ**: 「チャートの1本1本だよ。これ見てみろ」

```
    |  ← 上ヒゲ（高値）
  [緑] ← 陽線（上昇）: 始値より終値が高い
    |  ← 下ヒゲ（安値）

    |  ← 上ヒゲ
  [赤] ← 陰線（下降）: 始値より終値が低い
    |  ← 下ヒゲ
```

**ミコ**: 「1本のローソク足で、4つの情報が分かる」

```
1. 始値（開始時の価格）
2. 終値（終了時の価格）
3. 高値（期間中の最高価格）
4. 安値（期間中の最低価格）
```

#### 重要なパターン

**大陽線**: 長い緑のローソク → 強い買い圧力
**大陰線**: 長い赤のローソク → 強い売り圧力
**上ヒゲ長い**: 上昇を試したが失敗 → 下落の兆候
**下ヒゲ長い**: 下落したが反発 → 上昇の兆候

**ユウタ**: 「へー、これだけで色々分かるんだ」

**ミコ**: 「そう。じゃあ実践だ。SHIBのチャート見てみろ」

## 🎓 Lesson 2: RSI - 買われすぎ・売られすぎを見抜く

### ミコの教え

**ミコ**: 「次はRSI。Relative Strength Index、相対力指数だ」

**ユウタ**: 「相対...？」

**ミコ**: 「簡単に言うと、『買われすぎか、売られすぎか』を0-100の数字で表すもの」

```
RSI = 0〜100

【使い方】
70以上 → 買われすぎ（売りサイン）
30以下 → 売られすぎ（買いサイン）
50付近 → 中立
```

**ミコ**: 「例えば、RSIが25なら『売られすぎてる。そろそろ反発するかも』って判断できる」

**ユウタ**: 「なるほど！じゃあRSI 30以下のコインを探せばいいんだ！」

**ミコ**: 「そう。じゃあ実際に探してみろ。このシステムには戦略が組み込まれてる」

### 6時間後...

**ユウタ**: 「うわああああ！-15%！！」

```

```

**ミコ**: 「な？RSIだけじゃダメなんだ」

**ユウタ**: 「なんで!? RSI低かったのに！」

**ミコ**: 「**ダマシ**だよ。RSIが低くても、下降トレンドが強ければさらに下がる」

### ミコの解説

**ミコ**: 「テクニカル分析は100%じゃない。だから**複数の指標を組み合わせる**んだ」

```
【NG例】

【OK例】
RSI 28 AND ローソク足が反発の兆候 AND 出来高増加
→ 買い
```

**ユウタ**: 「損切りする...」

**ミコ**: 「それが正解。損失を最小限に抑えろ」

```

残高: 116円
```

**ユウタ**: 「また100円台に逆戻り...」

**ミコ**: 「落ち込むな。この失敗で『RSIだけじゃダメ』って学べただろ？**21円の授業料**だ」

## 🎓 Lesson 3: MACD - トレンドの方向を見極める

### ミコのリベンジ講座

**ミコ**: 「次はMACD。Moving Average Convergence Divergenceだ」

**ユウタ**: 「長っ...覚えられない...」

**ミコ**: 「名前はどうでもいい。要は『トレンドの方向と強さ』を見る指標だ」

#### MACDの見方

```
MACDライン
シグナルライン
この2本の位置関係で判断

【ゴールデンクロス】
MACDがシグナルを下から上に突き抜ける
→ 買いサイン！

【デッドクロス】
MACDがシグナルを上から下に突き抜ける
→ 売りサイン！
```

**ユウタ**: 「クロスするかどうか見ればいいんだ？」

**ミコ**: 「そう。しかもMACDは**ダマシが少ない**。RSIより信頼性が高い」

## 🎓 Lesson 4: ボリンジャーバンド - 大勝負の時

### ミコの最終講義

**ミコ**: 「Week 2最後の指標。ボリンジャーバンドだ」

**ユウタ**: 「ボリンジャー...？」

**ミコ**: 「価格の変動幅を視覚化するツール。こんな感じ」

```
      上限バンド (+2σ)
  ─────────────────
       移動平均線
  ─────────────────
      下限バンド (-2σ)
```

**ミコ**: 「価格は大体この範囲内で動く。で、**範囲を突き抜けた時が大チャンス**」

#### ボリンジャーバンドの戦略

```
【パターン1: 下限突破後の反発】
価格が下限を突き抜ける
→ 売られすぎ
→ 反発して上がる可能性大

【パターン2: スクイーズ後のブレイクアウト】
バンド幅が狭くなる（スクイーズ）
→ エネルギーを溜めてる状態
→ 突然爆発的に動く
```

**ユウタ**: 「爆発的！それいいね！」

**ミコ**: 「でも一番リスクも高い。大勝ちか大負けか、極端になる」
