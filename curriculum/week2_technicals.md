## 📚 学ぶこと

### 1. ローソク足の読み方

```
    |  <- 上ヒゲ（高値）
  [緑] <- 陽線（上昇）: 始値 < 終値
    |  <- 下ヒゲ（安値）

    |  <- 上ヒゲ
  [赤] <- 陰線（下降）: 始値 > 終値
    |  <- 下ヒゲ
```

**重要パターン**:
- **大陽線**: 強い上昇トレンド
- **大陰線**: 強い下降トレンド
- **上ヒゲ長い**: 上昇を試したが失敗→下落の兆候
- **下ヒゲ長い**: 下落したが反発→上昇の兆候

### 2. RSI（相対力指数）

**何を測る？**: 買われすぎ・売られすぎを判定

```
RSI = 0-100の値

70以上 → 買われすぎ（売りサイン）
30以下 → 売られすぎ（買いサイン）
```

**戦略例**:
```python
if RSI < 30:
    # 買いのチャンス
    buy_signal = True
elif RSI > 70:
    # 売りのチャンス
    sell_signal = True
```

### 3. MACD（移動平均収束拡散）

**何を測る？**: トレンドの方向と強さ

```
MACD線がシグナル線を上抜け → 買いサイン
MACD線がシグナル線を下抜け → 売りサイン
```

**ゴールデンクロス**: 強い買いサイン
**デッドクロス**: 強い売りサイン

### 4. ボリンジャーバンド

**何を測る？**: 価格の変動幅

```
        上限バンド (+2σ)
    ─────────────────
         移動平均線
    ─────────────────
        下限バンド (-2σ)
```

**戦略**:
- 価格が下限に触れる → 買いチャンス
- 価格が上限に触れる → 売りチャンス
- バンドが狭い → ブレイクアウトの予兆

## 🔬 実践課題

### 課題1: RSI戦略で取引
```
1. RSIが30以下の草コインを探す
2. 100円分購入
3. RSIが70超えたら売却
4. 結果を記録
```

### 課題2: MACD戦略で取引
```
1. MACDがゴールデンクロスした草コインを探す
2. 100円分購入
3. デッドクロスで売却
4. 結果を記録
```

### 課題3: ボリンジャーバンド戦略で取引
```
1. 価格が下限バンドに触れた草コインを探す
2. 100円分購入
3. 移動平均線まで戻ったら売却
4. 結果を記録
```

## 💡 今週の重要ポイント

1. **指標は100%ではない**: だましのシグナルもある
2. **複数の指標を組み合わせる**: RSI + MACDなど
