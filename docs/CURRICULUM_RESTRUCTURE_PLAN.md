### 問題点

#### 1. 内容の重複・矛盾

**Week 0 (00_foundations.md)**:
- ローソク足の読み方 ✓
- RSIの基礎 ✓
- MACDの基礎 ✓
- ボリンジャーバンドの基礎 ✓
- リスク管理 ✓

**Week 1 (01_start_and_strategy.md)**:
- 「Week 2でRSI」と記載 ← Week 0で既に学習済み
- 「Week 3でMACD」と記載 ← Week 0で既に学習済み
- テクニカル指標の説明なし ← Week 0で学習済みなのに活用していない

**Week 2-4 (02-04.md)**:
- RSI/MACD/BBの詳細解説 ← Week 0と重複

#### 2. 学習フローの不整合

```
【現状】
Week 0: 基礎知識（RSI, MACD, BB学習）
   ↓
Week 1: 取引開始（RSI/MACD使わず「なんとなく」取引）← 矛盾
   ↓
Week 2-4: RSI/MACD/BB詳細 ← Week 0と重複
```

**問題**: Week 0で学んだことがWeek 1で活用されていない

#### 3. カリキュラムの目的がブレている

- **旧カリキュラム**: 知識ゼロ → いきなり購入 → 失敗体験 → 後から学習
- **新カリキュラム**: 基礎知識 → 理解して購入 → データで検証

Week 1が旧カリキュラムのままなので、Week 0の意図が活かされていない

## 📊 新旧カリキュラム比較

### 旧カリキュラム（Week 0追加前）

```
Week 1: スタート → 何も知らずに購入 → 失敗
Week 2: RSI詳細 → 後から学ぶ
Week 3: MACD詳細 → 後から学ぶ
Week 4: BB詳細 → 後から学ぶ
Week 5-7: 高度な分析
```

**問題点**:
- 知識ゼロでいきなり購入 → リスク高い
- 失敗してから学習 → 非効率
- テクニカル指標を使わずに取引 → 根拠がない

---

### 新カリキュラム（Week 0追加後）

```
Week 0: 基礎知識 → RSI/MACD/BBの基礎を学ぶ
Week 1: 初めての購入 → Week 0の知識を活用して判断
Week 2: RSI実践 → 自分のデータで深掘り（Claude解説）
Week 3: MACD実践 → 自分のデータで深掘り（Claude解説）
Week 4: 統合分析 → 複数指標を組み合わせる
Week 5-7: 高度な分析 → ARIMA/GARCH/GRU
```

**改善点**:
- ✅ 知識を習得してから購入 → リスク軽減
- ✅ 理由を持って購入 → 根拠のある判断
- ✅ 実データで学習 → 理解が深まる
- ✅ Claude解説で個別化 → 動的カリキュラム
