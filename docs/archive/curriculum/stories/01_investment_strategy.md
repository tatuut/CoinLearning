## Scene 4: 3つの投資スタイル

**ミコ**: 「世の中には、3つの投資スタイルがある」

```markdown
## 1. スキャルピング（秒〜分）
- **狙い**: 小さな値動きで薄利多売
- **難易度**: ★★★★★（超高難度）
- **必要なもの**: 高速回線、板読みスキル、精神力
- **向いてる人**: プロトレーダー

## 2. デイトレード（分〜数時間）
- **狙い**: 1日の中での値動きを取る
- **難易度**: ★★★★☆（高難度）
- **必要なもの**: テクニカル分析、常時監視
- **向いてる人**: 時間がある人

## 3. スイングトレード（数日〜数週間）
- **狙い**: トレンドに乗って大きく取る
- **難易度**: ★★★☆☆（中難度）
- **必要なもの**: ファンダメンタルズ分析、忍耐力
- **向いてる人**: 兼業トレーダー

## 4. 長期投資（数ヶ月〜数年）
- **狙い**: 将来性に賭ける
- **難易度**: ★★☆☆☆（低難度）
- **必要なもの**: 信念、忍耐力
- **向いてる人**: ガチホ勢
```

## Scene 6: 「ユウタ式スイングトレード」の誕生

**ミコ**: 「じゃあ、俺たちの戦略を決める」

```markdown
# 【ユウタ式スイングトレード戦略】

## 🎯 目標
100円 → 1000円（10倍）

## ⏰ 期間
3-6ヶ月

## 📊 スタイル
スイングトレード（3日〜2週間保有）

## 🔍 銘柄選定
- 時価総額: Top 100-500（流動性あり、値動きあり）
- ボラティリティ: 中〜高（1日±3-8%）
- ニュース: ポジティブ材料あり
- トレンド: 上昇トレンド or 反転兆候

## 📈 エントリー条件（全て満たす）
1. **ファンダメンタル**: ポジティブニュースあり
2. **テクニカル**: 反発ポイント or ブレイクアウト
3. **センチメント**: 恐怖局面 or 熱狂初期

## 🚪 イグジット条件（どれか1つ）
1. **利確**: +20-30%到達
2. **損切り**: -10%到達
3. **期限**: 2週間経過（判断ミス）

## 💰 資金管理
- 1回の取引: 全資金の50-100%（少額のため）
- 連敗時: 3連敗で1週間休む
- 連勝時: 利益の一部を確保

## 📝 記録
- 全取引をメモ
- 勝因・敗因を分析
- 改善点を明確化
```

**ユウタ**: 「これが...俺たちの戦い方？」

**ミコ**: 「そう。**明確なルール**がある」

## Scene 7: 戦略の核心

**ミコ**: 「この戦略の核心は3つだ」

```markdown
## 【核心1: トリプル確認】

買う前に必ず3つ確認:
1. ファンダメンタルズ（なぜ上がるか？）
2. テクニカル（いつ入るか？）
3. センチメント（みんなどう思ってるか？）

→ 3つ揃わないと買わない
```

**ユウタ**: 「厳しくね？」

**ミコ**: 「チャンスは1週間に1-2回でいい。**確実な勝負だけする**」

```markdown
## 【核心2: 逆張りの美学】

みんなが恐怖 → 買い時
みんなが熱狂 → 売り時

ただし、ナイフが落ちてる時は拾わない
→ 底打ち確認してから
```

**ユウタ**: 「それってどう判断するの？」

**ミコ**: 「それを**技術**で解決する」

```markdown
## 【核心3: 理由なき撤退】

わからなくなったら即撤退
損切りラインを超えたら機械的に売る
感情を入れない
```

**ユウタ**: 「機械みたいだな」

**ミコ**: 「トレードは機械的にやるもんだ。感情は敵」

```markdown
判断基準:
1. ファンダメンタルズ: ポジティブ材料
2. テクニカル: 反発 or ブレイクアウト
3. センチメント: 恐怖 or 熱狂初期

利確: +20-30%
損切り: -10%

核心: トリプル確認、逆張り、機械的撤退
```

### 次に必要な「道具」

2. 反発ポイント検出
