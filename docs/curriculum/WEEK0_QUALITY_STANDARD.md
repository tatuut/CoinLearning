## 📊 各項目の品質基準

### 1. ローソク足の読み方

#### ❌ 不合格:
```
質問: 「この陽線は何を意味しますか？」
回答: 「...上がったってこと？」
```

#### ✅ 合格:
```
質問: 「この陽線は何を意味しますか？」
回答: 「終値が始値より高い。買いの勢いが強かった。
      上ヒゲが短いから、高値圏で終わってる。
      → 上昇トレンド継続の可能性」
```

**基準**:
- [ ] 陽線・陰線を見分けられる
- [ ] ヒゲの意味を説明できる
- [ ] 「この銘柄は今どう？」と聞かれて、ローソク足から判断できる

---

### 2. RSI（相対力指数）

#### ❌ 不合格:
```
質問: 「RSI 75の銘柄を買いますか？」
回答: 「...70以上だから買われすぎ？買わない？」
```

#### ✅ 合格:
```
質問: 「RSI 75の銘柄を買いますか？」
回答: 「買われすぎゾーン（70以上）。調整の可能性が高い。
      ただし、強いトレンドなら70以上が続くこともある。
      他の指標（MACD、BB）も確認してから判断する。
      → とりあえず今は様子見」
```

**基準**:
- [ ] RSI 70/30の意味を説明できる
- [ ] RSI単体では判断しない理由を理解している
- [ ] 「この銘柄のRSIは65です。どうする？」と聞かれて答えられる

**テスト問題**:
```
Q1: RSI 28の銘柄。買う？
A1: 売られすぎ（30以下）。反発の可能性あり。
    ただし、MACDがデッドクロスなら待つ。

Q2: RSI 80が1週間続いている。買う？
A2: 買われすぎだが、強いトレンド中の可能性。
    ニュースを確認。好材料があるなら
    トレンドフォロー。なければ調整待ち。
```

---

### 3. MACD（移動平均収束拡散）

#### ❌ 不合格:
```
質問: 「MACDがSignalを下から上に抜けた。これは？」
回答: 「...ゴールデンクロス？買い？」
```

#### ✅ 合格:
```
質問: 「MACDがSignalを下から上に抜けた。これは？」
回答: 「ゴールデンクロス。上昇トレンドの始まり。
      Histogramが拡大していれば勢いが強い。
      RSIが30以下ならさらに良い（売られすぎから反発）。
      → 買いシグナル。ただし、ボリンジャーバンドも確認」
```

**基準**:
- [ ] ゴールデンクロス/デッドクロスを説明できる
- [ ] Histogramの意味を理解している
- [ ] 「MACD < Signalです。どうする？」と聞かれて答えられる

**テスト問題**:
```
Q1: MACDデッドクロス発生。保有中の銘柄。どうする？
A1: 下降トレンド開始の可能性。
    RSI > 70なら損切り検討。
    RSI中立なら様子見。

Q2: Histogramが+0.01 → +0.05に拡大。意味は？
A2: 上昇トレンドが加速中。
    勢いが強まっている。
    → 保持継続、または買い増し検討。
```

---

### 4. ボリンジャーバンド

#### ❌ 不合格:
```
質問: 「価格が下限に接触。買う？」
回答: 「...売られすぎだから買う？」
```

#### ✅ 合格:
```
質問: 「価格が下限に接触。買う？」
回答: 「売られすぎの可能性。反発狙いで買い検討。
      ただし、バンドが縮小中（スクイーズ）なら
      大きな動きの前兆。方向がわからないので待つ。
      バンドが拡大中で、RSI < 30、MACDゴールデンクロスなら
      買いの好機」
```

**基準**:
- [ ] 上限/下限の意味を説明できる
- [ ] バンド幅の意味（スクイーズ/エクスパンション）を理解している
- [ ] 「価格が上限を突破した。どうする？」と聞かれて答えられる

**テスト問題**:
```
Q1: バンド幅が過去最小に縮小。どうする？
A1: スクイーズ。大きな動きの前兆。
    方向はまだわからない。
    → ゴールデンクロスやRSI 30以下を待つ。

Q2: 価格が上限を突破、バンド拡大中。どうする？
A2: 強い上昇トレンド（バンドウォーク）の可能性。
    RSI > 70でも買われすぎとは限らない。
    ただし、RSI 80超えなら調整待ち。
```

---

### 5. リスク管理

#### ❌ 不合格:
```
質問: 「損切りラインはどう決める？」
回答: 「...適当に？下がったら切る？」
```

#### ✅ 合格:
```
質問: 「損切りラインはどう決める？」
回答: 「購入前に決める。購入価格の-8〜-10%。
      例: 100円で購入 → 92円で自動損切り。
      感情で決めない。ルールを守る。
      → 損失を限定して、次のチャンスに備える」
```

**基準**:
- [ ] 損切りラインを購入前に決める理由を説明できる
- [ ] 分散投資の重要性を理解している
- [ ] 「総資金10,000円。1銘柄にいくらまで？」と聞かれて答えられる

**テスト問題**:
```
Q1: 1000円で購入。損切りラインは？
A1: 920円（-8%）または 900円（-10%）。
    購入前に決めて、守る。
    感情で「もう少し待てば...」は禁物。

Q2: 総資金 20,000円。5銘柄に分散。1銘柄いくらまで？
A2: 4,000円まで（20,000 ÷ 5）。
    1銘柄が失敗しても、他でカバーできる。
```

## 🎓 自己テスト

### テスト1: 総合判断

```
銘柄X:
- RSI: 42
- MACD: ゴールデンクロス発生（Histogram +0.02）
- BB: 下限から10%上（中央付近）
- ニュース: 特になし

Q: この銘柄を1000円で買いますか？
```

<details>
<summary>模範解答を見る</summary>

**判断: 買いOK**

理由:
1. **RSI 42**: 中立ゾーン。買われすぎではない ✅
2. **MACDゴールデンクロス**: 上昇トレンド開始 ✅
3. **BB中央付近**: 正常範囲内 ✅
4. **Histogram +0.02**: 上昇の勢いあり ✅

リスク管理:
- 損切りライン: 920円（-8%）
- 利確ライン: 1150円（+15%）
- 監視: 1日2回（朝・夜）

→ 総合的に買いシグナル。ただし、ポジションサイズは総資金の10-20%まで。
</details>

---

### テスト2: 危険回避

```
銘柄Y:
- RSI: 78
- MACD: デッドクロス発生
- BB: 上限突破
- ニュース: 「大きな発表なし」

Q: この銘柄を買いますか？
```

<details>
<summary>模範解答を見る</summary>

**判断: 買わない**

理由:
1. **RSI 78**: 買われすぎ（70以上） ❌
2. **MACDデッドクロス**: 下降トレンド開始 ❌
3. **BB上限突破**: 異常値 ❌
4. **3つの指標がすべて危険信号** ❌❌❌

もし保有中なら:
→ 損切り検討（RSI > 70 + デッドクロス）

買うタイミング:
→ RSIが60以下、MACDゴールデンクロスを待つ
</details>

## 💡 Week 0を効率よく学ぶコツ

```

✅ 良い例:
Week 0を読む

Week 1へ
```

**推奨サイト**: TradingView（無料）

**やること**:
1. BTC/USDTのチャートを開く
2. RSI、MACD、BBを表示
3. 「今、買う？売る？様子見？」を自分で判断
4. 理由を声に出して説明

**所要時間**: 10分 × 3日間 = 30分
