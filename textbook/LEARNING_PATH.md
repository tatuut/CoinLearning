**Week 1.5以降の教材は、実践の中で必要になった指標・概念を学びます。**


## 🔄 学習の進め方

### 基本サイクル

```
1. 📖 理論を学ぶ（Weekの教材）
    ↓
2. 💻 実践する（実際に取引）
    ↓
3. 📊 データで振り返る（ダッシュボード）
    ↓
4. 🎯 新しい課題を発見
    ↓
5. 📚 新しい概念を学ぶ
    ↓
1に戻る
```

### チェックリストの使い方

1. **学習前**: [CONCEPT_TREE.md](./CONCEPT_TREE.md)で未習得項目を確認
2. **学習中**: Weekの教材を進める
3. **学習後**: CONCEPT_TREE.mdにチェック ✅
4. **振り返り**: このファイル（LEARNING_PATH.md）で進捗確認
